**日付**: 2026年4月

## 要旨

ヴォイニッチ手稿（Beinecke MS 408）のEVA転写37,779語を対象に、仮説生成・仮説破壊の反復サイクルによる体系的計算分析を実施した。主要な発見は以下の4点である：(1) 接尾辞選択における統計的に有意な語間依存性（カイ二乗 = 1,404、p < 10^-278、Cramer's V = 0.215）で、ホールドアウトデータで交差検証済み；(2) 接頭辞と接尾辞が独立軸で機能する三項形態構造（NMI = 0.032）；(3) 既知のアルファベット体系と一致しないエントロピー減衰パターン（H1→H2で1.46ビット低下）；(4) イタリア語に最も近い統計的指紋（平均語長4.99）だが、異常に低いYule's K（28.33）。23の競合仮説を検証し、12を排除した。敵対的レッドチーム評価により信頼度を20-40ポイント引き下げ、中核的統計的発見は維持しつつ解釈的主張の限界を明示した。

### 1.1 仮説破壊プロトコル

各サイクルで仮説を生成し、次のサイクルで棄却を試みる。Cycle 16では専用の敵対的レッドチームが全ての主張を攻撃した。

## 2. 主要な発見

### 2.1 テキストは意味ある構造を持つ（ランダムではない）

| 指標 | ヴォイニッチ | 自然言語範囲 | 暗号範囲 |
|------|-------------|-------------|----------|
| TTR | 0.225 | 0.15-0.30 | 0.05-0.15 |
| 平均語長 | 4.99 | 4.0-5.8 | 可変 |
| 繰返率 | 0.00283 | 0.002-0.010 | 0.010-0.050 |
| Yule's K | **28.33** | 80-200 | 200-500+ |
| ハパックス比 | 0.695 | 0.50-0.60 | 0.70-0.80 |

語レベルの統計はイタリア語に合致するが、Yule's K（28.33）は既知の自然言語より著しく低く、規則的な語形成パターンを持つ構築されたシステムに整合する。

**レッドチーム評価後の信頼度**: テキストが回復可能な情報をエンコードしている確率55-65%。

### 2.2 形態構造：接頭辞 + 語幹 + 接尾辞

語は3つの位置スロットに分解される：

```
語 = [接頭辞] + 語幹/核 + [接尾辞]
```

- **接頭辞**: 15種（ch- 植物、d- 機能語、qo- 身体/星、ot- 天文、p- 項目マーカーなど）
- **接尾辞**: 17種（-am 終端子、-ol/-or 主語/話題開始、-aiin 属格、-y 述語/形容詞など）
- 接尾辞-amは73.2%が行末に出現 → 文/項目の終端子として機能

### 2.3 語間サンディ：新発見

本研究最大の統計的発見。n/l/rで終わる語（全体の46.4%、ほぼ均等に35:33:32%）は、後続語の頭文字と有意な相関を示す。

**三層条件付けシステム**:

| 層 | 要因 | 効果量 |
|----|------|--------|
| 第1層 | 語幹末母音 | MI = 0.929ビット（-i→-n 100%、-o→-l 79%、-a→-r 77%） |
| 第2層 | 後続語の頭文字 | Cramer's V = 0.215（母音前→-r 2.0倍、閉鎖音前→-l 1.4-2.1倍） |
| 第3層 | 韻律境界 | 行末→-n +5pt、行頭→-r +6pt |

**交差検証結果**:
- 50/50分割: 43.4% vs. 34.7%ベースライン（+8.7ポイント）
- 順列検定: p < 0.001（1,000回中0回が実精度に到達）
- 10分割CV: 42.9% +/- 2.7%
- 分割間確率相関: r = 0.84

**レッドチーム評価**: 相関は真実で再現可能。しかしCramer's V = 0.215は分散の約4%のみを説明する小さな効果。「サンディ」という用語は因果的音韻メカニズムを暗示するが、相関から因果は証明されない。EVA転写の人工物やCardan格子との区別は不完全。

**調整後信頼度**: 真の言語的特徴である確率60-70%。

### 2.4 フレーム復号：接頭辞=カテゴリ、接尾辞=役割

接頭辞と接尾辞の統計的独立性（NMI = 0.032）により、それぞれを分布的証拠から独立に復号可能。

**復号された接頭辞の例**:
- p-: 項目/段落マーカー（71.2%行頭、ページ冒頭語の45%）→ 信頼度HIGH
- d-: 機能語（daiinの「の」を含む）→ 信頼度HIGH
- ch-: 植物/薬草用語 → 信頼度HIGH
- cth-: 特定の植物属性（70%が薬草セクション、行内後半位置0.60）→ 信頼度HIGH
- ot-: 天文学的実体（天文セクションで2.5倍過剰表現）→ 信頼度HIGH

**復号された接尾辞の例**:
- -am: 文終端子（73%行末）→ 信頼度VERY HIGH
- -ol: 主語/主格マーカー（71%行頭偏向）→ 信頼度HIGH
- -y: 述語/形容詞マーカー（23%行末、最大接尾辞）→ 信頼度HIGH
- -aiin: 属格マーカー（daiin=「の」を含む）→ 信頼度HIGH

### 2.5 接頭辞-接尾辞の独立性（NMI = 0.032）

完全分節語（n = 17,761）の相互情報量分析：

| ペア | MI（ビット） | NMI |
|------|-------------|-----|
| 接頭辞-語幹 | 1.653 | 0.508（相関あり） |
| 語幹-接尾辞 | 1.372 | 0.379（相関あり） |
| **接頭辞-接尾辞** | **0.104** | **0.032（独立）** |

自然言語では接頭辞-接尾辞NMIは0.15-0.40。0.032は多次元記法体系の特徴。

### 2.6 文字体系：音節文字的・特徴的、標準アルファベットではない

文字レベルのエントロピー減衰:

| 次数 | ヴォイニッチ | イタリア語 | ラテン語 |
|------|-------------|-----------|---------|
| H(1) | 3.87 | 3.90 | 4.00 |
| H(2) | **2.41** | 3.30 | 3.50 |
| H(3) | 2.17 | 2.60 | 2.80 |

H(1)→H(2)の1.46ビット低下は欧州アルファベット言語の約3倍。文字バイグラムが極めて予測可能であることを示し、音節文字・アブギダ・構造化記法体系に整合する。

295個の異なる文字バイグラムはイタリア語の音節目録（200-300）とほぼ一致。

### 2.7 統計的指紋：イタリア語に最も近い

多次元距離分析の結果：

| 順位 | 言語/システム | 距離 |
|------|-------------|------|
| 1 | アラビア語（古典） | 0.789 |
| 2 | イタリア語散文（14世紀） | 0.909 |
| 3 | イタリア語（科学） | 1.038 |
| 9 | 単純置換暗号 | 2.019 |
| 10 | ランダムテキスト | 2.277 |

### 2.8 植物名コード：77%がユニーク初語

薬草フォリオの冒頭語分析（f1r-f57r）：

- **56フォリオ中43（77%）**がセクション全体で唯一の語で開始
- **53/56（95%）**が3回以下の出現
- ランダム選択の期待値: 5-10%

→ 各フォリオの冒頭語は植物名をエンコードしていると強く示唆。

**アンカー同定**:
- f2r: kydainy = Paeonia（シャクヤク、55%）
- f3r: tsheos = Rubia（アカネ、60%）
- f9r: tydlo = Nigella（クロタネソウ、65%）
- f41r: psheykedaleey = Adiantum（ホウライシダ、60%）
- f47r: pchair = Vitis（ブドウ、70%）

### 2.9 ディオスコリデスの巻レベル順序

5つのアンカー同定に基づく重み付き線形回帰：

- **巻レベルの進行が確認**: f1r-f11r → 第3巻、f13r-f49r → 第4巻、f50r-f57r → 第5巻
- **章レベルの予測は完全に失敗**: 非アンカー10件中0件が図解と一致
- 巻間の語彙は体系的に変化（Jaccard類似度 0.12-0.14）

### 2.10 Davis のフォリオ再配列の確認

Lisa Fagin Davis（2020-2025）の写本学的研究が示した現在のフォリオ順序の誤りを、テキスト分析が独立に確認：

- Davis が隣接と特定したフォリオ対（f84v/f78r）が最高の語彙一致スコア（Jaccard = 0.2371）
- 共役葉（同一バイフォリオ）の一致度が連続葉より19%高い（0.1070 vs. 0.0898）
- 共役葉間の語彙分離度は平均90.7% → 異なる筆写者が交互にバイフォリオを筆写

## 4. 生存モデル：二層記法体系

### 4.1 構造

```
語 = [接頭辞]    +    語幹    +    [接尾辞]
       |                |              |
  カテゴリ          概念           役割
  （15種）        （開集合）     （17種）
       |                               |
       +------ 独立 ------------------+
               (NMI = 0.032)
```

**第1層（復号済み -- フレーム）**: 15のカテゴリ接頭辞と17の文法接尾辞が領域と機能をエンコード。
**第2層（未復号 -- コア）**: 約200-400の語幹が特定の概念（植物名、物質、調合法）をエンコード。

### 4.2 歴史的文脈

15世紀初頭の北イタリア（パドヴァ-ヴェネツィア軸）に合致：
- ヴェネツィア外交暗号（nomenclator）伝統（1401-1450年）
- Giovanni Fontanaのパドヴァにおける暗号手稿（1418-1430年代）
- 偽ルルスの薬学的錬金術文書
- ヴェネツィアのテリアク貿易と十人委員会の毒物研究
- 錬金術的Decknamen（植物のコード名、約160件）

## 7. 結論

本研究はヴォイニッチ手稿について、敵対的レビューに耐える4つの事実を確立した：

1. **テキストは真の語間統計的依存性を示す**（p < 0.001、偶然から21標準偏差）。これは文献で未報告の新発見である。

2. **語は三項構造に分解される**（接頭辞 + 語幹 + 接尾辞）。接頭辞と接尾辞は統計的に独立（NMI = 0.032）で、直交軸上でエンコードされる。

3. **文字レベルのエントロピー減衰**（H(1)→H(2)で1.46ビット低下）は既知のアルファベット文字体系と不一致であり、音節文字・アブギダ・構造化記法体系を示唆する。

4. **現在のフォリオ順序は原本ではない**。写本学的証拠（Davis）とテキスト的証拠（語彙一致度）の独立した収束により確認。

ヴォイニッチ手稿は依然として未解読である。本研究の貢献は解読ではなく、構造的特性の厳密な特徴付け、12の仮説の体系的排除、そして解読に必要なものの明確な仕様化である：すなわち、さらなる暗号解析ではなく、核心語彙を構成する約200の語幹の特定 -- 15世紀北イタリアの薬学実践に関する領域専門知識を要する課題 -- が必要である。
