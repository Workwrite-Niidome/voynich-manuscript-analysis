# Voynich Manuscript: Agglutinative Morphology Comparative Analysis

## 1. Agglutinative Languages with PREFIX+ROOT+SUFFIX Case Systems

The Voynich pattern — PREFIX + ROOT + SUFFIX, with case particles formed by PREFIX + -aiin — is unusual. Most agglutinative languages are purely suffixing. Below is a systematic comparison.

### Turkish (Eliminated)
- **Structure:** Purely suffixing. ROOT + suffix + suffix + suffix...
- **Case system:** 6 cases, all suffixes (-i, -e, -de, -den, -in, -le)
- **Match to Voynich:** POOR. No prefixing at all. Despite being the classic agglutinative exemplar, Turkish cannot produce the PREFIX+ROOT+SUFFIX pattern observed in Voynichese.

### Finnish / Hungarian (Eliminated)
- **Structure:** Purely suffixing, like Turkish
- **Case system:** Finnish has 15 cases, Hungarian has 18 — all suffixes
- **Match to Voynich:** POOR. Same problem as Turkish: no prefixes in nominal morphology.

### Georgian (Strong Match)
- **Structure:** PREFIX + ROOT + SUFFIX (both prefixing and suffixing)
- **Case system:** 7 cases (nominative, ergative, dative, genitive, instrumental, adverbial, vocative)
- **Key feature:** Polypersonal verb agreement — a single verb can encode subject, direct object, and indirect object via combinations of prefixes and suffixes
- **Circumfixes:** Georgian uses circumfixes (e-...-eb for desideratives), meaning a prefix and suffix jointly create a grammatical meaning — neither is meaningful alone
- **Verb template:** Up to 8 morphemes in one verb: preverb + agreement prefix + version vowel + ROOT + thematic suffix + tense suffix + agreement suffix
- **Match to Voynich:** STRONG. Georgian's prefix+root+suffix structure, polypersonal agreement, and circumfix usage closely parallel the Voynich pattern. The PREFIX+-aiin pattern could be analogous to Georgian circumfixes where the prefix modifies what the suffix grammaticalizes.

### Basque (Moderate Match)
- **Structure:** Primarily suffixing for nouns, but verb forms use both prefixes and suffixes
- **Case system:** Ergative-absolutive; ergative -k/-ek, absolutive zero-marked, dative -i
- **Key feature:** Periphrastic verb forms with auxiliary agreement; agglutinative suffixation for nominal morphology
- **Match to Voynich:** MODERATE. Basque nouns are purely suffixing (poor match), but the verbal morphology uses both prefixes and suffixes (better match). The case system itself is too simple (3 core cases) compared to the multiple d-aiin/s-aiin/k-aiin forms in Voynichese.

### Sumerian (Strongest Match)
- **Structure:** PREFIX CHAIN + ROOT + SUFFIX CHAIN — both nominal and verbal
- **Nominal morphology:** Noun root (typically 1-2 syllables) + case suffix. But noun phrases allow modifiers between root and case marker.
- **Verbal morphology:** Modal prefix + conjugation prefix + dimensional prefix + ROOT + aspect suffix + agreement suffix. Up to ~15 morphological slots in a verb chain.
- **Case markers:** Absolutive (-zero), ergative (-e), dative (-ra/-e), genitive (-ak), locative (-a), comitative (-da), equative (-gin), directive (-she), ablative (-ta)
- **Key feature:** Biconsonantal roots dominate. The verb root is almost always monosyllabic. Noun roots are typically one or two syllables.
- **Match to Voynich:** STRONGEST.
  - 2-consonant/monosyllabic roots = matches Voynich's 2-consonant root dominance
  - Prefix chain + root + suffix chain = matches PREFIX+ROOT+SUFFIX
  - Case suffixes attached to phrase-final position = could explain -aiin as a phrase-level case marker
  - Agglutinative with clear morpheme boundaries = matches Voynich's regular morphology

### Ranking: Best Match to Voynich PREFIX+ROOT+SUFFIX Pattern

| Rank | Language | Prefix+Root+Suffix | Biconsonantal Roots | Case Complexity | Overall |
|------|----------|-------------------|---------------------|-----------------|---------|
| 1 | **Sumerian** | Yes (verb chain) | Yes (dominant) | 9+ cases | Excellent |
| 2 | **Georgian** | Yes (circumfixes) | No | 7 cases | Strong |
| 3 | **Basque** | Partial (verbs only) | No | 3 core cases | Moderate |
| 4 | Turkish | No (suffix only) | No | 6 cases | Poor |
| 5 | Finnish/Hungarian | No (suffix only) | No | 15-18 cases | Poor |

**Critical observation:** Sumerian is the only well-documented language that combines (a) prefix+root+suffix morphology, (b) biconsonantal/monosyllabic root dominance, and (c) agglutinative case marking. However, Sumerian was extinct millennia before the Voynich manuscript's creation (early 15th century). This raises the question: could the Voynich system be *modeled on* a Sumerian-type template, or could it reflect a still-unknown language with similar typological properties?

## 2. Languages with Both Sandhi AND Agglutination

The Voynich shows phonological alternation at word/morpheme boundaries (sandhi) alongside agglutinative structure. This combination is typologically specific.

### Sanskrit
- **Sandhi:** Extremely elaborate. Both internal (within words) and external (between words). Vowel coalescence, consonant assimilation, visarga alternation.
- **Agglutination:** Partial. Sanskrit is primarily fusional/inflectional (Indo-European), but has agglutinative features in compound formation (samasa) and some verbal prefixing.
- **Match:** MODERATE. Rich sandhi but not truly agglutinative — it's fundamentally fusional. The sandhi rules are well-documented and very regular, which matches Voynich regularity.

### Dravidian Languages (Tamil, Kannada, Malayalam, Telugu)
- **Sandhi:** Complex and extensive. Three major types in Kannada: Lopa (elision), Aagama (insertion), Aadesh (substitution). Tamil has vowel sandhi with consonant glide insertion and external sandhi that can become morphologized.
- **Agglutination:** Strong. Words formed by concatenating mostly invariant suffixes, each with a single grammatical function. Morpheme boundaries are generally clear.
- **Key insight:** In Dravidian languages, agglutination *causes* sandhi — the process of concatenating morphemes triggers phonological adjustments at boundaries. This is exactly what the Voynich pattern suggests.
- **Match:** STRONG. The Dravidian combination of regular agglutination + boundary-triggered sandhi is the closest typological parallel to Voynich morphophonology.

### Kannada (Specific)
- **Sandhi types:** Vowel elision (lopa), consonant insertion (aagama), sound replacement (aadesh)
- **Word formation:** Compounds + phonotactics (sandhi) + prefixation + serial suffixation
- **Morphological richness:** Highly agglutinative with sandhi as an integral grammatical aspect
- **Match:** STRONG. Kannada specifically uses prefixation alongside suffixation, making it one of the few Dravidian languages that approaches the Voynich PREFIX+ROOT+SUFFIX pattern while maintaining sandhi.

### Other Languages with Sandhi + Agglutination
- **Sinhala:** Sandhi + agglutinative features (Indo-Aryan with Dravidian substrate influence)
- **Pali:** Sandhi rules similar to Sanskrit, more regular morphology
- **Korean:** Morphophonemic alternations at morpheme boundaries + agglutinative structure
- **Japanese:** Rendaku (sequential voicing) at compound boundaries + agglutinative morphology

### Summary Table: Sandhi + Agglutination

| Language | Sandhi Complexity | Agglutination Level | Prefix+Suffix | Voynich Match |
|----------|------------------|---------------------|---------------|---------------|
| Tamil | High | High | Suffix-dominant | Moderate |
| Kannada | High | High | Some prefixing | Strong |
| Sanskrit | Very High | Moderate (fusional) | Both | Moderate |
| Korean | Moderate | High | Suffix-only | Weak |
| Japanese | Low-Moderate | High | Suffix-only | Weak |

## 3. 15th-Century Contact Languages and Pidgins

### Lingua Franca / Sabir (Mediterranean Pidgin)
- **Period:** 11th-19th century, peak use in 15th-16th centuries
- **Lexicon:** 65-70% Italian (heavy Venetian and Ligurian), 10% Spanish, plus Arabic, Catalan, Greek, Occitan, Sicilian, Turkish
- **Grammar:**
  - Radically simplified: no singular/plural distinction
  - Verbs used in infinitive/unconjugated form for all persons
  - Past tense via periphrastic "mi estar andato" construction
  - SVO word order
  - Minimal inflection
- **Match to Voynich:** POOR for direct identification (Sabir is too simplified to produce the complex morphology seen in Voynichese). However...

### Could the Voynich be a Pidgin/Creole Pharmaceutical Language?

**Arguments FOR:**
1. The extremely regular morphology could reflect a *deliberately simplified* system designed for cross-cultural communication
2. A pharmaceutical pidgin would explain the botanical/medical content with systematic notation
3. The 15th-century Mediterranean was a zone of intense multilingual contact (Italian, Arabic, Greek, Turkish, Hebrew traders all exchanging pharmaceutical knowledge)
4. The regularity of Voynichese morphology exceeds natural language variation — pidgins and creoles are known for regularizing grammar
5. Pharmaceutical knowledge was traded across language barriers; a standardized notation would have practical value

**Arguments AGAINST:**
1. Pidgins typically simplify, not complexify — the Voynich PREFIX+ROOT+SUFFIX system is more complex than typical pidgin grammar
2. Known pidgins (including Sabir) lack agglutinative morphology
3. The systematic case-marking system (d-aiin, s-aiin, etc.) suggests a *natural* case system, not a pidgin simplification
4. No historical evidence of a pharmaceutical pidgin with this level of grammatical structure

**Verdict:** A *pure* pidgin is unlikely, but a **specialized register or jargon** built on an agglutinative base language is plausible. This would be a natural language used in a restricted domain (pharmacy/botany) with conventionalized forms.

## 4. Constructed Pharmaceutical Notation Systems

### Known 15th-Century Systems

#### Alchemical Notation
- Medieval alchemy developed graphic symbols for substances and processes
- The 15th century was a key period for expanding alchemical terminology, including vernacular introduction
- Symbols were compact notations, not a full language — they encoded substances (mercury, sulfur, gold) and operations (distillation, calcination), not grammar
- **Match to Voynich:** POOR. Alchemical notation is symbolic/logographic, not phonetic/morphological. The Voynich script behaves like a phonetic writing system, not a symbol system.

#### Apothecary Abbreviation Systems
- Specific symbols for measurement: ounce (like a flat-topped "3"), scruple (backwards "e"), dram (from "yogh")
- "ss" for "semis" (half), "Rx" for "recipe" (take), "ana" for equal quantities
- System: pound = 12 ounces, ounce = 8 drachms, drachm = 3 scruples, scruple = 20 grains
- Recipes written in Latin with standardized abbreviations
- **Match to Voynich:** POOR for the writing system itself, but INTERESTING structurally. The apothecary system shows that pharmaceutical professionals already used specialized notation. The Voynich could represent a *more ambitious* version — a full encoding system rather than just abbreviations.

#### Scribal Abbreviation in Medical Manuscripts
- Medieval medical manuscripts used extensive abbreviations transferred from Latin
- Specific medical abbreviations for common terms, measurements, and instructions
- Standardized across European pharmacopeias
- **Match to Voynich:** The Voynich is too complex and too systematic to be an abbreviation system. It has full sentence-like structures, not just abbreviated recipes.

### Could the Voynich be a Purpose-Built Notation?

**The Constructed Language Hypothesis:**

The evidence points toward a system that is *more regular than any natural language* but *more complex than any known notation system*. This suggests one of two possibilities:

1. **A natural agglutinative language** written in an unfamiliar script — the regularity reflects the language's own morphological regularity (agglutinative languages are inherently more regular than fusional ones)

2. **A constructed/engineered language** designed for pharmaceutical/botanical encoding — the regularity reflects deliberate design

**Evidence favoring construction:**
- Morphological regularity exceeds natural language norms
- The PREFIX+-aiin case system is suspiciously systematic
- No known 15th-century language matches all features simultaneously
- The manuscript's content (plants, recipes, astronomical/astrological data) suggests a specialized purpose

**Evidence favoring natural language:**
- Statistical properties (word frequency distributions, entropy) match natural language
- The system is too complex for a constructed notation (why encode grammar if you just need to label plants?)
- Contact-zone languages can exhibit unusual combinations of features
- Agglutinative languages (especially Dravidian, Georgian) naturally produce highly regular morphology

## 5. The PREFIX+-aiin Pattern: Grammaticalization Analysis

### What is PREFIX+-aiin?

The Voynich case particles d-aiin, s-aiin, k-aiin, r-aiin, t-aiin appear to consist of a meaningful consonantal prefix (d-, s-, k-, r-, t-) combined with a grammaticalizing base -aiin. This is a circumfix-like pattern where the prefix specifies the *type* of grammatical relation and -aiin marks the word as a *grammatical particle*.

### Parallels in Known Languages

#### Georgian Circumfixes
- **Pattern:** e-...-eb (prefix + root + suffix forming a single grammatical meaning)
- **Example:** e-cekv-eb-a "he feels like dancing" (circumfix e-...-eb = desiderative)
- **Parallel:** The Voynich d-...-aiin could function like a Georgian circumfix where d- specifies case type and -aiin marks particle status

#### Sumerian Dimensional Prefixes + Case Suffixes
- **Pattern:** Prefix chain specifies relationships, suffix marks case
- **Example:** mu-na-n-sum "he gave it to him" — mu (ventive) + na (dative) + n (ergative) + sum (give)
- **Parallel:** Sumerian uses prefixes for relational meaning and suffixes for grammatical category, exactly like Voynich PREFIX+-aiin

#### Berber Circumfixes
- **Pattern:** t-...-t marks feminine gender
- **Example:** t-argaz-t (feminine of argaz "man")
- **Parallel:** A discontinuous morpheme where prefix and suffix jointly encode a grammatical feature

#### German ge-...-t (Past Participle)
- **Pattern:** ge- prefix + -t suffix = past participle
- **Example:** ge-mach-t "made"
- **Parallel:** A circumfix where neither ge- nor -t alone creates the past participle meaning

#### Malay/Indonesian Circumfixes
- **Pattern:** Multiple circumfixes: ke-...-an, pe-...-an, ber-...-an
- **Example:** ke-besar-an "greatness" (ke-...-an = nominalizer around root besar "big")
- **Parallel:** Different prefixes + same suffix = different grammatical functions. This is the closest structural parallel to Voynich d/s/k/r/t + -aiin.

### The Malay Parallel is Striking

In Malay/Indonesian:
- **ke-...-an** = stative/abstract nominalizer
- **pe-...-an** = process/action nominalizer
- **ber-...-an** = reciprocal/distributed action

The PREFIX varies to indicate *type of grammatical function*, while the SUFFIX (-an) remains constant as the grammaticalizing marker. This is structurally identical to:
- **d-...-aiin** = case function D
- **s-...-aiin** = case function S
- **k-...-aiin** = case function K

This suggests -aiin is a **grammaticalizing suffix** (like Malay -an) that creates function words, while the prefix consonant (d-, s-, k-, r-, t-) specifies *which* grammatical function.

### Grammaticalization Pathway

The diachronic cycle of grammaticalization typically proceeds:
1. Content word (free morpheme with lexical meaning)
2. Clitic (phonologically bound but syntactically free)
3. Affix (morphologically bound)
4. Fusion/loss

If -aiin was originally a content word meaning something like "relation" or "connection," and various consonantal prefixes were originally independent words meaning "from," "to," "at," "with," "by," the system would represent **Stage 2-3**: the content words have become bound morphemes (cliticized/affixed) but haven't yet fused.

This is consistent with either:
- A **young agglutinative language** still in early grammaticalization
- A **constructed language** that froze the system at a convenient stage
- A **pidgin/creole** that stabilized grammaticalization patterns from a source language

## 6. Synthesis: What Language Type Best Fits the Voynich?

### Feature Checklist

| Feature | Sumerian | Georgian | Dravidian | Malay | Constructed |
|---------|----------|----------|-----------|-------|-------------|
| PREFIX+ROOT+SUFFIX | Yes | Yes | Partial | Yes (circumfix) | Designable |
| Biconsonantal roots | Yes | No | No | No | Designable |
| Sandhi at boundaries | Unknown | Minor | Yes | No | Designable |
| Case via PREFIX+suffix | Yes (verb) | Yes (circumfix) | No | Yes (circumfix) | Designable |
| Extreme regularity | Moderate | Moderate | Moderate | Moderate | Yes |
| 15th c. availability | No (extinct) | Yes | Yes | Yes | Yes |
| Vowel insertion in roots | Yes | Yes | Yes | Yes | Designable |

### Top Hypotheses (Ranked)

1. **Constructed language modeled on agglutinative principles** — Best explains the extreme regularity, the systematic PREFIX+-aiin circumfixes, and the lack of exact match with any known language. The designer may have been familiar with multiple language types (possibly including Dravidian, Turkic, or Caucasian languages encountered through trade).

2. **Unknown Caucasian language** (Georgian family or neighboring) — Georgian-type prefix+suffix morphology with circumfixes is the strongest natural-language parallel. The Caucasus had many undocumented languages in the 15th century, some now extinct.

3. **Dravidian-influenced register** — A specialized pharmaceutical vocabulary built on a Dravidian (Kannada-like) base would explain sandhi + agglutination + moderate prefix use. Indian pharmaceutical knowledge was transmitted to the Mediterranean through Arabic intermediaries.

4. **Sumerian-inspired notation** — If the author had access to cuneiform scholarship (unlikely but not impossible through Arabic scholarly traditions that preserved some Mesopotamian knowledge), the Sumerian template of prefix chain + monosyllabic root + suffix chain could have inspired the system.

5. **Austronesian (Malay-type) circumfix system** — The PREFIX+-aiin pattern maps almost perfectly to Malay ke-/pe-/ber- + -an circumfixes. However, Austronesian contact with 15th-century Europe was minimal (pre-Portuguese exploration of the East Indies).

### The Strongest Single Conclusion

The Voynich morphological system most closely resembles a **circumfix-based agglutinative language** where:
- The ROOT carries lexical meaning (plant name, action, substance)
- The PREFIX specifies relational/grammatical type
- The SUFFIX (-aiin and others) grammaticalizes the whole unit into a specific function

This pattern is attested in Georgian, Malay, Berber, and (for verbs) Sumerian — but no known 15th-century European language exhibits it. The extreme regularity suggests either an undocumented natural language or deliberate construction.
