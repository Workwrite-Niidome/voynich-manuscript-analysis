## 1. Visual Description of the 9-Rosette Foldout

### 1.1 Overview

The 9-rosette foldout (page 158 in the Beinecke PDF scan) is a six-panel fold-out page -- the largest single diagram in the entire Voynich manuscript. When fully unfolded, it reveals **9 circular medallions (rosettes)** arranged in a roughly **3x3 grid**, connected by tubes, channels, and causeways. The page is oriented in landscape format when unfolded.

The overall arrangement is:

```
+------------------+--------------------+------------------+
|                  |                    |                  |
|  [NW Rosette]    |   [N Rosette]      |  [NE Rosette]   |
|  (ring pattern)  |   (blue petal      |  (irregular,    |
|                  |    flower/star)    |   organic)       |
|                  |                    |                  |
+--------+---------+--------+----------+--------+---------+
|        |                  |                   |         |
| [W     |   [CENTER ROSETTE]                   | [E      |
| Rosette|   (largest, dominant)                | Rosette]|
| (small |   (concentric rings with             | (green  |
|  ring) |    buildings/structures)              |  tint)  |
|        |                                      |         |
+--------+---------+--------+----------+--------+---------+
|                  |                    |                  |
|  [SW Rosette]    |   [S Rosette]      |  [SE Rosette]   |
|  (organic/wave   |   (blue 8-petal    |  (cross/tower   |
|   pattern)       |    rosette)        |   structure)    |
|                  |                    |                  |
+------------------+--------------------+------------------+
```

### 1.2 Individual Rosette Descriptions

**CENTER ROSETTE (largest)**
- Position: Dead center of the foldout, significantly larger than the others
- Design: Multiple concentric rings surrounding a central oval or lens-shaped element
- Content: The innermost area contains what appears to be an architectural structure -- possibly a walled city, castle, or complex of buildings with battlements
- The concentric rings contain dense text and small decorative elements (possibly trees or vegetation)
- A scalloped or crenellated outer border suggests fortification
- This is the dominant hub of the entire diagram

**NORTH ROSETTE (top center)**
- Position: Top center, medium size
- Design: A striking **blue-petaled flower or star** at center (approximately 8 petals/rays)
- The petals radiate from a central point, colored in blue/cyan pigment
- Surrounded by a circular band of text
- Connected to the center rosette by a tube/causeway descending downward

**NORTHWEST ROSETTE (top left)**
- Position: Upper left
- Design: A **ring or donut pattern** -- a concentric circle with a clear center
- Contains what appears to be a smaller ring within a larger ring
- Text visible along the circumference
- Connected to adjacent rosettes by causeways/tubes

**NORTHEAST ROSETTE (top right)**
- Position: Upper right
- Design: More **irregular and organic** than the others -- possibly depicting terrain, hills, or a landscape
- Less geometrically regular than other rosettes
- May contain small buildings or structures within organic forms

**WEST ROSETTE (middle left)**
- Position: Left center, relatively small
- Design: A **small ring pattern**, possibly with a central element
- Connected to the center rosette by a horizontal causeway
- Appears to have a simpler internal structure than the center

**EAST ROSETTE (middle right)**
- Position: Right center
- Design: Filled with a **greenish tint** or wash
- Contains what may be landscape elements or vegetation
- Connected to the center rosette

**SOUTHWEST ROSETTE (bottom left)**
- Position: Lower left
- Design: **Organic, wave-like patterns** -- possibly depicting water, terrain, or flowing forms
- Contains curving lines suggesting natural features (rivers? coastlines?)
- Small structures or towers visible within

**SOUTH ROSETTE (bottom center)**
- Position: Bottom center, medium size
- Design: A prominent **blue 8-petal rosette/flower** pattern, similar to but distinct from the north rosette
- Blue pigment is one of the most vivid colors on the entire page
- Geometric and symmetrical

**SOUTHEAST ROSETTE (bottom right)**
- Position: Lower right
- Design: Contains what appears to be a **cross or tower structure**
- Possibly a Maltese cross or similar architectural/religious symbol
- Small marks that could be text or decorative elements

### 1.3 Connections Between Rosettes

The rosettes are connected by **tubes, channels, or causeways** -- narrow strips with parallel borders that create pathways between the circular elements. Key observations:

- The causeways are **not mere lines** -- they have width and often contain interior detail (parallel lines suggesting a road or channel)
- Some causeways appear to flow INTO the rosettes (entering through breaks in the circumference)
- The center rosette is the **hub** -- most causeways connect to it directly or indirectly
- The overall impression is of a **network** rather than a simple list
- Some causeways appear to have text alongside them

### 1.4 Text Placement

Based on the EVA transcription, text appears in several distinct positions:
- **Within the rosettes**: Blocks of paragraph text (especially f85r1 with 35 lines, f86v6 with 45 lines)
- **Along circular borders**: Circumscription text following the rosette edges
- **Along causeways**: Short labels or descriptions along connecting paths (visible in the `<->` notation in the transcription, indicating text spans across a dividing line)
- **In the interstices**: Text in the spaces between rosettes

## 2. Llullian Wheel Comparison

### 2.1 Structural Parallels

| Feature | Llullian Figures | 9-Rosette Foldout |
|---------|-----------------|-------------------|
| **Number of primary elements** | 9 (Dignities B-K) | 9 (rosettes) |
| **Arrangement** | Circular (in a single wheel) | Circular elements in a grid |
| **Central element** | Sometimes (the Fourth Figure has a center) | Yes (center rosette is largest) |
| **Connecting structures** | Triangles, lines between sectors | Causeways/tubes between circles |
| **Concentric rings** | Yes (2-3 rotating circles) | Yes (center rosette has multiple rings) |
| **Sectors/divisions** | Yes (9 sectors per circle) | Some rosettes show internal divisions |
| **Color coding** | Red, yellow, green in pseudo-Lullian | Blue, green, earth tones |
| **Text in circles** | Letters B-K as labels | Voynichese text in blocks and along borders |

### 2.2 Concentric Ring Structures

The **center rosette** clearly shows concentric ring structures, which is the defining feature of Llullian wheels:

- Llull's First Figure: A single circle divided into 9 sectors
- Llull's Fourth Figure: 3 concentric rotating circles, each with 9 sectors
- The center rosette: Multiple concentric rings (at least 3-4 visible) surrounding a central built structure

The concentric structure of the center rosette is the strongest visual parallel to Llullian wheels in the entire manuscript.

### 2.3 Sector Divisions

Several rosettes show **internal divisions** reminiscent of wheel sectors:
- The N and S rosettes have **8-petal star/flower patterns** -- dividing the circle into 8 sectors radiating from center
- The center rosette's concentric rings may contain sector divisions (difficult to determine at this resolution)
- The NW rosette shows a ring-within-ring pattern suggesting concentric zoning

However, the Voynich rosettes differ from pure Llullian wheels in that they contain **figurative content** (buildings, landscapes, water) rather than abstract letters. This is more consistent with the **pseudo-Lullian alchemical tradition**, which added illustrative content to the wheel framework.

### 2.4 The 8-Petal Pattern

The N and S rosettes display **8-petal rosette patterns** in blue. This is noteworthy because:
- 8 petals could represent the **8 directions** (cardinal + ordinal)
- Or they could represent **8 qualities** (the 4 Galenic qualities x 2 degrees)
- In some Llullian manuscripts, **8 + 1 = 9** appears as a structural pattern (8 peripheral + 1 central)
- The foldout itself uses this pattern: 8 peripheral rosettes + 1 central = 9 total

## 3. Text Extraction: Rosette Section Analysis

### 3.1 Folio-to-Rosette Mapping

The EVA transcription divides the rosette text into 6 labeled sections. Based on the standard Voynich folio numbering and the transcription's position markers ($P= codes), these correspond to different rosettes:

| Section | Lines | Position Code | Likely Rosette |
|---------|-------|---------------|----------------|
| f85r1 | 35 lines | $P=B | One of the large rosettes (perhaps NW or center top half) |
| f85r2 | 24 lines | $P=C | Adjacent rosette |
| f86v4 | 9 lines | $P=N | Smaller rosette (fewer lines suggest less text area) |
| f86v6 | 45 lines | $P=O | Largest text section -- likely the center rosette |
| f86v5 | 39 lines | $P=P | Large rosette |
| f86v3 | 36 lines | $P=Q | Large rosette |

**Key observation**: f86v6 has the most text (45 lines), consistent with it being the CENTER ROSETTE (the largest). f86v4 has only 9 lines, consistent with one of the smaller peripheral rosettes.

### 3.2 Text Type Annotations

The transcription uses position codes that reveal text placement:
- `@P0` / `+P0` = Paragraph text (main body)
- `@Pb` / `+Pb` = Text in band/border (circumscription)
- `@Cc` = Text along circular border or causeway
- `@Pc` / `+Pc` = Short text, possibly captions

This reveals that the rosette pages contain **three types of text**:
1. **Paragraph blocks** (P0) -- extended descriptions within rosettes
2. **Band text** (Pb) -- text running along circular borders, circumscribed
3. **Causeway/connection text** (Cc) -- text along the tubes connecting rosettes

### 3.3 Circumscription Text (Band Text, Pb)

The `Pb` (band) annotations are particularly Llullian. In Llullian wheels, text runs along the circular bands of the concentric circles. The Voynich rosettes similarly have text that follows circular paths:

**f85r2 band text** (7 blocks, short lines):
```
sain.or.or.aiin.opchdy
qotor.sheedy.shodaiin.olfar.ary
dair.sheo.oraiin.chol.daiin
ockhdor.olkor.shoral
sosees
```

**f86v3 band text** (multiple blocks):
```
tchedy.qokeedy.qoedy.qopchy.raiin.olpchey.qokaiin
sain.ychedy.qoeeey.otair.cheody.ytain.qokaiin
```

These short, label-like text blocks running along circular borders closely parallel how Llullian wheel sectors are labeled.

### 3.4 Causeway Text (Cc)

The `Cc` annotation appears in f85r2 and f86v4, marking text along connecting structures:

**f85r2.1** (long continuous line):
```
odchdy.otedy.opoees.ar.chcthy.otchdy.otedy.otar.chepaiin.otadar.otodaiin...
```

**f86v4.1-4** (paired lines with `<->` separator):
```
opor.os.sheody.pchedar.odsheo.qody... <-> qopcher.cher.oseeedar.odaiin...
oeeey.o?daiin.otedaiin.otedy... <-> chotaiin.oraiin.otodeeeo.or...
```

The `<->` notation in f86v4 is remarkable: it indicates text that spans ACROSS a dividing line, with words on both sides of a causeway or connecting tube. This is exactly how a Llullian connecting diagram would work -- labels on both sides of a line that links two categories.

### 3.5 Distinctive Vocabulary

Comparing the rosette sections' vocabulary to the rest of the manuscript:

**High-frequency terms in rosette text**:
- `qokaiin`, `qokar`, `qotar` (qo- prefix cluster) -- extremely common in rosette text
- `ykaiin`, `ytaiin`, `ytar` (y- prefix cluster) -- very frequent
- `olkar`, `olkaiin`, `olkeedy` (ol- compounds) -- frequent
- `otar`, `otaiin`, `otal` (ot- prefix cluster)

**Observation**: The rosette text shows heavy use of the `qo-` and `y-` prefixes, which in our previous analysis were identified as **specialized/restricted categories** (73-80% paradigm completeness vs. 100% for broad categories). This is consistent with the rosettes encoding specialized categorical information rather than general herbal descriptions.

## 4. Numerical Structure Analysis

### 4.1 The Number 9

There are **exactly 9 rosettes** on the foldout page. This is the single most striking numerical parallel with Llull's system:

- Llull used **9 Dignities** (Bonitas, Magnitudo, Aeternitas, Potestas, Sapientia, Voluntas, Virtus, Veritas, Gloria) = B through K
- Llull's First Figure has **9 sectors**
- Llull's Fourth Figure has **9 letters per concentric circle**
- The Voynich foldout has **9 rosettes**

The probability of this being coincidence depends on how common the number 9 is in medieval diagrammatic traditions. While 9 appears in various contexts (9 Muses, 9 celestial spheres, 9 angelic orders), the specific combination of 9 circular elements with connecting structures in a single diagram is distinctively Llullian.

### 4.2 Internal Divisions

- **N and S rosettes**: 8-petal patterns (8 sectors + center = 9 total? Or simply 8?)
- **Center rosette**: Multiple concentric rings (appears to be 3-4)
- **Paragraph blocks per section**: f85r1 has 8 paragraph markers (<%>), f86v6 has 5, f86v5 has 6, f86v3 has 6

The number of paragraph divisions (marked by <%>) within each rosette section:

| Section | Paragraph blocks (<%>) | Lines |
|---------|----------------------|-------|
| f85r1 | 9 blocks | 35 |
| f85r2 | 4 blocks | 24 |
| f86v4 | 2 blocks | 9 |
| f86v6 | 5 blocks | 45 |
| f86v5 | 6 blocks | 39 |
| f86v3 | 5+ blocks | 36 |

f85r1 has **9 paragraph blocks** -- matching the number of rosettes. This could indicate that f85r1's text describes or refers to all 9 rosettes sequentially.

### 4.3 The 8+1 Pattern

The rosette layout follows an **8+1 pattern**: 8 peripheral rosettes surrounding 1 central rosette. This mirrors several Llullian structures:

- Llull's First Figure: 9 sectors that can be read as 8 peripheral concepts + 1 unifying center
- The pseudo-Lullian *Liber de secretis naturae*: Uses diagrams with a central figure surrounded by peripheral elements
- Medieval cosmological diagrams: Earth at center, surrounded by 8 celestial spheres (though usually 7 or 9)

### 4.4 Fibonacci / Other Patterns

The text volumes across sections (35, 24, 9, 45, 39, 36 lines) do not show obvious Fibonacci patterns. However, the total text (~188 lines) across 6 transcribed sections suggests that either:
- 3 rosettes have no text (purely visual)
- Some rosettes share transcription sections
- The transcription covers only portions of each rosette

## 5. Reinterpretation: The Rosettes as a Combinatorial Diagram

### 5.1 The Map Hypothesis (Traditional)

The traditional interpretation of the 9-rosette foldout as a **geographical map** identifies specific locations:
- Some researchers have proposed it shows cities in Northern Italy or the Eastern Mediterranean
- The "causeways" are interpreted as roads or trade routes
- Buildings within rosettes represent actual cities/fortifications
- The center could be a particular important city

### 5.2 The Combinatorial Diagram Hypothesis (Llullian)

Under the Llullian framework established in our previous analysis, the 9-rosette foldout can be reinterpreted as:

**A visual representation of how 9 pharmaceutical categories interact and combine.**

Each rosette would represent one of 9 fundamental categories (paralleling Llull's 9 Dignities), and the causeways between them would represent **combination rules** -- which categories can be combined, and how.

**Proposed mapping to pharmaceutical categories:**

| Rosette Position | Possible Category | Llullian Dignity Parallel | Visual Evidence |
|-----------------|-------------------|--------------------------|-----------------|
| **CENTER** | Master category / Universal solvent | (Central principle) | Largest, most complex, architectural |
| **N** | Celestial/Planetary influence | Bonitas (Goodness) | Blue star/flower pattern |
| **NW** | Mineral substances | Magnitudo (Greatness) | Ring pattern (like a crucible?) |
| **NE** | Wild/gathered herbs | Aeternitas (Duration) | Organic, landscape-like |
| **W** | Distillation processes | Potestas (Power) | Small, concentrated ring |
| **E** | Growing/cultivated herbs | Sapientia (Wisdom) | Green tint, vegetation |
| **SW** | Aquatic/water-based preparations | Voluntas (Will) | Wave/water patterns |
| **S** | Compound medicines | Virtus (Virtue) | Blue rosette pattern |
| **SE** | Institutional/religious preparations | Veritas/Gloria | Cross/tower structure |

### 5.3 Causeways as Combination Rules

If the causeways represent combination rules, their structure would indicate:
- **Which categories can be combined**: Connected rosettes = compatible categories
- **Direction of combination**: If causeways have directional flow (into vs. out of rosettes)
- **Type of combination**: The text along causeways (Cc annotations) could specify the combination method

The `<->` notation in f86v4's transcription is particularly suggestive: it shows text on both sides of a connecting line, exactly as a Llullian combination table pairs two categories.

### 5.4 Buildings as Institutional Categories

The buildings/structures within rosettes (especially the center) could represent **institutional contexts** for pharmaceutical practice:
- **Walled city/castle** (center): The main pharmacy or hospital
- **Towers**: Storage facilities or specialized preparation rooms
- **Organic/garden areas**: Herb gardens (physic gardens)
- **Water features**: Baths or distillation facilities

This interpretation aligns with the medieval pharmaceutical tradition where different institutions had different roles:
- The **apothecary** prepared medicines
- The **hospital** administered them
- The **garden** grew raw materials
- The **library** stored recipes and knowledge

### 5.5 The Text as Categorical Output

The distribution of text across rosette sections supports the combinatorial interpretation:

- **f86v6** (45 lines, likely center rosette): The longest text, potentially describing the master category or providing an overview of all combinations
- **f85r1** (35 lines, 9 paragraph blocks): May enumerate one item from each of the 9 categories
- **f86v4** (9 lines): A minimal entry, perhaps for a restricted or simple category
- The variation in text length mirrors how different Llullian categories produce different numbers of valid combinations

## 6. Comparison with Specific Llullian Diagrams

### 6.1 Llull's First Figure (Figura A)

Llull's First Figure is a **single circle divided into 9 sectors**, each labeled with a letter (B-K) and a Dignity. Lines connect each sector to every other sector, creating a web of relationships.

**Parallel with the rosette foldout**: The 9 rosettes connected by causeways create a similar **web of relationships** between 9 categories. The key difference is that the Voynich separates the 9 elements into distinct circles rather than sectors of one circle, and uses tubes/causeways rather than inscribed lines for connections.

This is what you might expect if someone **expanded** Llull's compact First Figure into a more elaborate visual format, giving each category its own dedicated space for illustration and description.

### 6.2 Llull's Second Figure (Triangles)

The Second Figure inscribes **3 triangles** in a circle, encoding 9 relational principles in groups of 3 (difference/concordance/contrariety; beginning/middle/end; majority/equality/minority).

**Parallel**: The rosette foldout's 3x3 grid arrangement creates implicit groupings of 3:
- Top row (NW, N, NE)
- Middle row (W, Center, E)
- Bottom row (SW, S, SE)

Each row of 3 could represent a Llullian triad of related concepts, with the causeways showing how triads interact.

### 6.3 Llull's Fourth Figure (Rotating Wheels)

The Fourth Figure has **3 concentric rotating circles**, each bearing the letters B-K. The center rosette of the Voynich foldout shows **concentric rings** that visually echo this structure.

If the center rosette IS a representation of the Fourth Figure's combinatorial output, the surrounding 8 rosettes could show the **results** of specific combinations -- each peripheral rosette depicting what happens when particular categories are combined.

### 6.4 Pseudo-Lullian Alchemical Diagrams

The pseudo-Lullian *Liber de secretis naturae* contains a **Tractatus septem rotarum** (Treatise of Seven Wheels, of which six rotate). This work:
- Uses **multiple interconnected circular diagrams** (not just one)
- Applies the wheel system to **alchemical/pharmaceutical processes**
- Was produced in the **same geographical and temporal context** as the Voynich (Northern Italy, 14th-15th century)

The 9-rosette foldout's structure of **multiple interconnected circles** is closer to the pseudo-Lullian multi-wheel tradition than to Llull's own single-circle figures. This is exactly what we would expect if the Voynich author was working within the pseudo-Lullian pharmaceutical tradition rather than with Llull's original theological works.

### 6.5 The "Arbor" (Tree) Tradition

Both Llull and the pseudo-Lullian tradition used **tree diagrams** (arbores) to show hierarchical classification. The rosette foldout's causeway network could be read as a **flattened tree** or **network graph** showing how categories branch and connect.

The center rosette as "root" with causeways branching to 8 peripheral rosettes mirrors the arbor structure with a trunk (center) and branches (connections to periphery).

## 7. Key Findings and Assessment

### 7.1 Strongest Parallels

1. **The number 9**: The foldout has exactly 9 rosettes, matching Llull's exactly 9 Dignities. This is the most numerically precise parallel.

2. **Concentric rings in the center rosette**: The center rosette's concentric ring structure visually echoes Llullian wheel construction.

3. **The 8+1 pattern**: 8 peripheral + 1 central mirrors Llullian structures where one element is conceptually primary.

4. **Circumscription text (Pb annotations)**: Text running along circular borders is characteristic of Llullian wheel labeling.

5. **Causeway text with `<->` notation**: Text spanning both sides of connecting tubes directly parallels how Llullian combination tables pair categories.

6. **Multiple interconnected circles**: Closer to the pseudo-Lullian *Tractatus septem rotarum* tradition than to any map tradition.

### 7.2 Important Differences

1. **Figurative content**: Llullian wheels contain abstract letters/categories; the Voynich rosettes contain buildings, landscapes, and natural features. However, pseudo-Lullian manuscripts DO add figurative elements.

2. **Grid vs. wheel**: Llull arranges 9 elements as sectors of ONE circle; the Voynich uses 9 SEPARATE circles. This is an expansion, not a contradiction.

3. **No visible letter labels**: Llullian wheels prominently display the letters B-K. The Voynich rosettes use Voynichese text instead. Under our hypothesis, the morphemic notation REPLACES the letter system.

4. **Architectural content**: The buildings and structures within rosettes have no Llullian precedent but DO have precedent in medieval mapping and institutional classification traditions.

### 7.3 Synthesis

The 9-rosette foldout is most parsimoniously interpreted as a **hybrid diagram** combining:

1. **Llullian categorical structure** (9 categories, interconnected, with a combinatorial framework)
2. **Pseudo-Lullian multi-wheel elaboration** (multiple interconnected circles rather than one wheel)
3. **Pharmaceutical institutional mapping** (buildings representing types of preparation or storage)
4. **The Voynich's own morphemic notation** (replacing Llull's letter system with the prefix+root+suffix code)

The foldout likely served as a **visual key** to the entire manuscript's organizational scheme -- showing the reader the 9 fundamental categories and how they relate, before the detailed text (in the surrounding recipe and herbal sections) provides the specific combinatorial outputs.

### 7.4 Confidence Assessment

| Claim | Confidence |
|-------|-----------|
| The foldout has exactly 9 rosettes | **Confirmed** (visible in image) |
| The center rosette has concentric ring structure | **Confirmed** (visible in image) |
| The rosettes are connected by tube-like causeways | **Confirmed** (visible in image) |
| The structure parallels Llullian wheel diagrams | **Strong** (70-80%) |
| The 9 rosettes correspond to Llull's 9 Dignities | **Moderate** (50-60%) -- numerically exact but content mapping is speculative |
| The causeways represent combination rules | **Moderate** (45-55%) -- consistent but unverifiable without decipherment |
| The foldout is a pharmaceutical combinatorial diagram rather than a map | **Moderate** (50-60%) -- more consistent with manuscript context but map hypothesis also viable |
| The center rosette represents Llull's Fourth Figure | **Speculative** (30-40%) -- visual parallel exists but stretched |

### 7.5 What Would Confirm or Refute This Interpretation

**To confirm:**
- If the text within each rosette uses different prefix distributions (each rosette = different category), the combinatorial interpretation gains strength
- If rosettes with causeways between them share vocabulary that rosettes WITHOUT direct connections do not share, the "combination rule" interpretation is supported
- If the 8-petal patterns in N and S rosettes divide into exactly 8 labeled sectors matching any known 8-fold pharmaceutical classification

**To refute:**
- If the buildings can be conclusively identified as specific real-world cities/locations, the map interpretation prevails
- If the text within different rosettes shows no vocabulary differentiation (all using the same prefix distributions), the categorical interpretation fails
- If the number of connecting causeways does not match any combinatorial pattern (e.g., if every rosette connects to every other, that's 36 connections -- matching Llull's Third Figure -- vs. selective connections suggesting a different organizing principle)
