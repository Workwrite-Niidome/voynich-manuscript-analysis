## 6. Collocation-Based Validation

### Test: dar = "cum/con" (with)?

**Evidence FOR**:
- Uniformly distributed (CV=0.39) -- "with" is needed everywhere
- avg_pos=0.56 -- mid-to-late clause, where "with [ingredient]" appears
- 13% line-initial -- can start a new instruction ("with X, do Y")
- Followed by: ar(10), chedy(6), aiin(6), ol(6), chey(5), oty(5) -- a mix of function and content words, consistent with "with + [noun phrase]"
- Pharmaceutical rate (14.3/k) is above average -- recipes heavily use "with"

**Evidence AGAINST**:
- Not especially recipe-concentrated (recipe rate 4.5/k is below herbal 9.5/k)
- Uniformity might indicate "and/but" rather than "with"

**Verdict**: dar likely = **"et" (and)** rather than "cum (with)". Its extreme uniformity (the MOST uniform of all top-10 words) and position profile better match the most basic conjunction. However, daiin already maps to "et" by frequency. Alternative: dar = **"sed/ma" (but)** -- the contrasting conjunction, which should also be fairly uniform and clause-medial.

### Test: or = "cum/con" (with)?

**Evidence FOR**:
- Pharmaceutical-heavy (26.9/k) -- exactly where "with" matters most
- Followed overwhelmingly by -aiin words (49 times by aiin alone) -- "with + [noun]" pattern
- avg_pos=0.46 -- mid-clause, where "cum [ablative]" appears
- 9% line-initial -- can begin recipe instructions

**Verdict**: or = **"cum/con" (with)** -- STRONG. The pharmaceutical concentration is exactly right for a recipe-heavy "with" preposition.

### Test: chol = "hic/haec" (this)?

**Evidence FOR**:
- Herbal-concentrated (20.8/k herbal vs 1.8/k cosmo) -- "this [plant]" is constant in herbal entries
- avg_pos=0.43 -- early-to-mid clause, where demonstratives appear
- Strongly followed by: daiin(28), chol(21), chy(8), cthol(7), chor(7) -- followed by grammatical words building a noun phrase
- 5% line-initial

**Verdict**: chol = **"hic/haec/questa" (this)** -- STRONG. The herbal concentration is diagnostic.

### Test: shey = "quod/qui" (which/that)?

**Evidence FOR**:
- 68% followed by content words -- highest rate of any major function word -- introduces a new clause about a specific thing
- Cosmological concentration (17.9/k) -- astronomical descriptions use relative clauses heavily
- avg_pos=0.45 -- mid-clause, typical for relative pronouns

**Verdict**: shey = **"qui/quod/che" (which/that)** -- MODERATE-STRONG. The high content-after rate is distinctive.

### Test: am = "est" (is)?

**Evidence FOR**:
- 68.6% line-final -- verbs/copulas end Latin clauses
- avg_pos=0.86 -- overwhelmingly late in lines
- Universal presence across sections
- Only 102 occurrences -- moderate frequency matches "est"

**Evidence AGAINST**:
- Frequency seems low for "est" (would expect higher)

**Verdict**: am/dam/otam = **verb/copula complex**. The -am suffix may encode a verb ending. dam (d+am) = "it is of", otam (ot+am) = verbal form.

## 8. Structural Model Summary

The 309 empty-stem words (33% of corpus) form a complete grammatical system:

```
SENTENCE = [OPENER] + NOUN_PHRASE + [CONNECTOR] + NOUN_PHRASE + [TERMINATOR]

OPENER (s-, t-, p- prefixes):
  saiin, sar, sol, sor, sain, sair  = "then/next/thus"
  taiin, tar, tol, tor, tain, tair  = "take/use" (imperative)

CONNECTOR (d-, ch-, sh- prefixes, bare suffixes):
  daiin  = "and" (et)
  dar    = "but" (sed)  
  or     = "with" (cum)
  aiin   = "in" (in)
  ol     = "of/from" (de)
  ar     = "to" (ad)
  al     = "for" (per)

DETERMINER (ch- prefix):
  chol   = "this" (hic)
  chey   = "[copula] is" or "the"

RELATIVE (sh- prefix):
  shey   = "which/that" (qui/quod)

TERMINATOR (-am, -ary, -dy suffixes):
  am/dam/otam  = verb ending / "is"
  dy/ody/chdy  = adjectival ending
  ary/dary     = manner/quality ending
```

### The PREFIX encodes grammatical FUNCTION:
- **d-** = basic relational (prepositions, conjunctions)
- **s-** = sequential/procedural (recipe steps)
- **t-** = imperative/instructional
- **ch-** = demonstrative/nominal
- **sh-** = relative/descriptive
- **ok-** = object/patient case
- **ot-** = verbal/processual
- **qo-** = domain-specific determiner

### The SUFFIX encodes grammatical CATEGORY:
- **-aiin/-ain** = nominal (genitive/accusative)
- **-ar/-al** = directional/purposive
- **-or/-ol** = locative/ablative
- **-ey/-eey** = descriptive/adjectival
- **-y/-dy** = terminal/completive
- **-am/-om** = verbal (copula/imperative)
- **-an** = terminal case

This system is **agglutinative**: 18 prefixes x 27 suffixes = 486 possible combinations, of which 309 are attested. The empty-stem words are the pure grammatical morphemes -- function words constructed from the same building blocks as content words, but without lexical stems.

## 9. Cross-Validation Checks

### Check 1: daiin as "et" (and)
- Frequency: 712 = 1.91% of corpus. Latin "et" typically 2-4% of pharmaceutical text. **CONSISTENT**.
- Position: avg 0.50, fairly even. "Et" appears everywhere. **CONSISTENT**.
- Collocation: Preceded by chol(28), ol(14), chey(10) -- comes after noun phrases. **CONSISTENT**.
- Followed by: chey(14), shey(13), cthy(10) -- begins new noun phrases. **CONSISTENT**.

### Check 2: or as "cum" (with)  
- Frequency: 341 = 0.92%. Latin "cum" typically 0.5-1.5%. **CONSISTENT**.
- Section: Pharma 26.9/k (highest); recipes use "cum" heavily. **CONSISTENT**.
- Followed by: aiin(49), ain(10), aiiin(10), al(9) -- "with + noun" pattern. **CONSISTENT**.
- or+aiin = "cum+[noun-genitive]" = "with the [X]". **CONSISTENT**.

### Check 3: chol as "hic" (this)
- Frequency: 360 = 0.97%. "Hic/haec/hoc" typically 0.5-1.5%. **CONSISTENT**.
- Section: Herbal 20.8/k >> Cosmo 1.8/k. "This plant" dominates herbal entries. **CONSISTENT**.
- Followed by: daiin(28), chol(21-self-repeating), chy(8). "This [X] and..." **CONSISTENT**.
- chol+chol (21 occurrences): "this and this" or demonstrative reinforcement. **PLAUSIBLE**.

### Check 4: am as verbal ending
- Position: avg 0.86, 69% line-final. Latin verbs end sentences. **CONSISTENT**.
- d+am = "dam" (58, 66% line-final): "d-" prefix + verbal ending = "is [of]" or imperative. **CONSISTENT**.
- ot+am = "otam" (43, 63% line-final): "ot-" (verbal prefix) + verbal ending. **CONSISTENT**.
- ch+am = "cham" (24, 58% line-final): nominal + verbal = "[it] is the...". **CONSISTENT**.

### Check 5: Clause structure test
A typical Voynich line should read as:
```
[s-opener] [content-word] [d-connector] [ch-determiner] [content-word] [or-preposition] [content-word] [-am terminator]
```

Parsing: ycheor / chol(this) / odaiin / chol(this) / s / aiin(in) / okolor / am(is)
"[Plant-name] this [X]-of this [Y] in [Z]-colors is"
-- Reads as a coherent botanical description. **CONSISTENT**.

### The 33% finding
That fully one-third of the manuscript consists of pure prefix+suffix combinations is itself a major structural finding. This ratio is unusually high for natural language but consistent with:
1. An agglutinative constructed notation system
2. A language with a very productive morphological system and many function words
3. A pharmaceutical text genre that is heavily formulaic with repeated grammatical patterns

The empty-stem words are not random -- they show clear syntactic positioning (line-initial s-/t- words, line-final -am words), section-specific distributions (ch- herbal, qo- cosmological), and systematic collocation patterns. Whatever the Voynich manuscript encodes, its grammatical system is real and structured.
