# ヴォイニッチ手稿の形態論的構造と検証済み制約条件: 敵対的テストを生き残った知見

**著者:** Anonymous (方向性と監督), Claude Opus 4.6 (計算分析), Anthropic

**コーパス:** EVA転写 RF1b-e.txt (37,779トークン, 8,500ユニーク語型, 226フォリオ)

**日付:** 2026年4月 (v2 -- ブラインド予測テスト後の改訂版)

---

## 概要

ヴォイニッチ手稿(ベイネッケMS 408, 放射性炭素年代測定1404-1438年)は1世紀以上にわたり解読を拒んできた。本研究では、EVA転写全体の体系的計算分析を、仮説生成--敵対的仮説破壊の反復サイクルで実施し、レッドチームレビューと2回の事前登録ブラインド予測テストを含む。

本報告書は、すべての失敗を成功と同等に報告する。23の仮説を検証し、12を棄却した。

---

## 第1部: 検証済みの知見（全テストを生存）

### 1.1 語間サンディ（接尾辞依存性）

ヴォイニッチ手稿の語末子音(n/l/r、ほぼ等確率)の選択が、後続語の語頭文字によって体系的に条件付けられる。

- **10-fold交差検証**: 42.9% +/- 2.7% (ベースライン34.7%に対しすべてのフォールドで上回る)
- **順列テスト**: p < 0.001 (1000回中0回が実精度に到達)
- **分割間相関**: Pearson r = 0.84 (p = 5.36 x 10^-13)
- **行内vs行間**: 行内で作用するが行境界で崩壊（韻律的単位に一致）

この知見は単純なカルダングリルでは再現困難であり、Rugg (2004)のいたずら仮説を実質的に弱体化させる。

### 1.2 形態論的構造

```
WORD = [PREFIX] + [CLASSIFIER + ROOT + VOWEL GRADE + TERMINAL] + [SUFFIX]
```

- **接頭辞**: 15の生産的タイプ (ch- 16.9%, qo- 14.2%など)
- **接尾辞**: 17タイプ（行位置による機能分化）
- **接頭辞--接尾辞の独立性**: NMI = 0.032（自然言語の0.15-0.40に比べ極めて低い）

この独立性は、自然形態論よりも多次元記法体系に整合的。

### 1.3 母音グレード = 特異性 (r = -0.954)

母音グレードによる派生連鎖(k -> ke -> kee -> keeo)がある。3仮説を検証:

| 仮説 | 結果 |
|------|------|
| H1: 植物部位 | **棄却** -- パターンが予測に矛盾 |
| H2: 処理段階 | 支持されるが特異性と交絡 |
| H3: 特異性 | **強く支持** -- r = -0.954, R^2 = 0.910 |

ベアステムは多くのフォリオに出現する一般カテゴリ(平均3.14フォリオ/語型)。高グレードはより限定的な文脈に制限される(+eeoで1.90フォリオ/語型)。レシピセクションは精度を要求するため、高グレードの体系的濃縮(+ee 2.82x, +eeo 3.82x)を示す。

### 1.4 複合分類子の合成性 (8/8)

8つの複合語根(kch, tch, pch, lch, lsh, ckh, cth, cph)すべてが、構成要素の意味から予測されるセクション分布を示した。

### 1.5 形態素のページレベル分布 (14/15がBonferroni補正を生存)

15の形態素のうち14が、1,000回のランダムシャッフルと比較して極めて高いページレベル分散を示す(Z = 3.15-29.51)。6,384の可能な部分文字列に対するBonferroni補正後も生存。

**重要な注意**: 非ランダム分布は意味的内容を証明しない。CurrierのA/B言語差異、筆記者クラスタリング、主題的語彙変動でも説明可能。

### 1.6 検証済み形態素: ty = 細い/線形の葉 (p = 0.008)

112の本草フォリオ全体で検証した5つの形態素-特徴連関のうち、唯一有意に複製されたもの:

| 指標 | 値 |
|------|-----|
| Fisher正確検定 p | 0.008 |
| オッズ比 | 18.0 |
| 感度 | 0.250 (線形葉ページの25%に出現) |
| 特異度 | 0.982 (広葉ページの98%に不在) |
| 陽性的中率 | 0.800 (tyが出現すれば80%が線形葉ページ) |

## 第2部: 棄却された主張と失敗したアプローチ

### 2.1 sh = 地下/根: 棄却

**ブラインドテスト結果**: 
- 陽性予測(sh出現 -> 根が目立つ): 0.5/2 = 25% (**偶然以下**)
- 陰性予測(sh不在 -> 根が目立たない): 43.8% (**偶然以下**)

**コーパスの反証**: sholは根が目立つフォリオ(0.99%)より葉が目立つフォリオ(1.42%)で高頻度。予測の逆方向。

**誤りの原因**: ch=地上部に対する構造的対称性の仮定（データではなく美学に基づく）。反証が存在したが合理化された。

**修正解釈**: sh-語はqok-(量的マーカー)に1.93x高率で後続される。sh=物質/成分マーカーの可能性。ただし未検証。

### 2.2 cthy = 果実/種子: 全規模で棄却

- **元の研究**: 9ページ訓練セットでchi2 = 60.44
- **112ページ全体**: Fisher p = 0.661, OR = 1.29
- **原因**: 訓練セットの選択バイアス。元の9ページがcthy高頻度のサブセットだった

### 2.3 dchy = 高い/樹木状: 棄却

- **元の研究**: 3ページでchi2 = 32.36
- **112ページ全体**: Fisher p = 1.0, OR = 1.22（信号なし）
- **原因**: 3ページサンプルの4語変動による人為的結果

### 2.4 陰性予測の体系的失敗

両ブラインドテストで、「形態素が不在なのでその特徴も不在」という予測が偶然レベル以下:

| 予測タイプ | 精度 | 偶然ベースライン |
|-----------|------|----------------|
| oiin不在 -> 花なし | 43% | ~45% |
| yd不在 -> 分裂葉なし | 50% | ~65% |
| dal不在 -> 丸い特徴なし | 58% | ~70% |
| sh不在 -> 目立つ根なし | 43.8% | ~50% |

### 2.5 その他の棄却された仮説

- 音声復元（5つのアンカー植物名すべて失敗）
- 古トルコ語仮説（母音調和58%、偶然レベル）
- Bax部分音声体系
- -yd- = 分裂葉の4/4一致（多重検定で予想される偽陽性）
- ディオスコリデス比較スコア（ベースラインなし）

### 2.6 小サンプルからの過大な統計値が複製されない理由

元の研究は3-9ページのサブグループで稀な語(6-82出現)のカイ二乗値を計算。これは極めて不安定:

- dchy: 3ページ・4語でchi2 = 32.36 -> 22ページに拡大すると信号ゼロ
- cthy: 9ページでchi2 = 60.44 -> 29ページに拡大するとOR = 1.29

**教訓**: 小Nの言語学研究における稀な形態素の印象的なカイ二乗値は、確立された知見として扱うべきではない。

## 第3部: 有望な仮説（未検証）

### 3.1 記述的命名体系

形態論的構造、語根の合成性、図版相関から、語が観察可能な特徴を符号化する記述的命名体系が示唆される。

**ブラインドテスト2の陽性予測結果**:

| 陽性予測 | 精度 | 偶然ベースライン |
|---------|------|----------------|
| cth出現 -> 果実/種子 | 83% (2.5/3) | ~35% |
| dal出現 -> 丸い/球状 | 100% (2/2) | ~30% |
| oiin出現 -> 花 | 100% (1/1) | ~55% |
| ty高 -> 細い/線形 | 100% (1/1) | ~25% |
| **合計** | **~79%** | **~35%** |

合計二項p値は約 < 0.01。ただしN = 9.5は個別の有意性には不足。

**ステータス**: 大規模事前登録テスト(N > 30)が必要。検証済みとは言えない。

### 3.2 ch/sh機能的対比

sh=根の棄却後、コーパス分析で:
- sh-語はqok-(量的マーカー)に1.93xで後続
- sh-語はsh-語に連鎖（成分リスト）
- sh/比率はライン1(見出し)で2.23x高い

ch = 視覚的記述 vs sh = 物質/成分という機能的対比が示唆される。未検証。

## 第4部: ブラインド予測テスト -- 方法論的貢献

### 4.1 意義

知る限り、ヴォイニッチ研究で事前登録ブラインド予測テストを実施した先行研究はない。これは本研究の重要な方法論的貢献である。

### 4.2 テスト1 (10フォリオ: f20r-f38r)

- **全体精度**: 67.6% (37予測中25ヒット+パーシャル)
- **推定偶然**: ~60-65%
- **重要結果**: sh=根が決定的に棄却された(25%)

### 4.3 テスト2 (10フォリオ: f7r-f19r)

- **全体精度**: 63.2% (57予測中36ヒット+パーシャル)
- **推定偶然**: ~55-60%
- **重要結果**: 陽性予測79% vs 陰性予測50%の非対称性を発見

### 4.4 テストから得られた洞察

1. **陽性予測は機能する; 陰性予測は機能しない** -- 部分記述子モデルに制約
2. **一部の形態素意味は誤り** -- sh=根がクリーンに棄却
3. **小N訓練セットは膨張した統計を生む** -- 5形態素中4が全規模で複製失敗
4. **ベースレートが重要** -- 「葉が目立つ」は常にyesで~70%達成可能

## 第5部: レッドチーム結果

### 攻撃に耐えた主張:
1. 形態素分布は真に非ランダム (テストB: 14/15, Z = 3.15-29.51)
2. テキストは方向性構造を持つ (テストD: 反転でパターン崩壊)
3. ランダム意味割当は分解を再現しない (テストF: 0/50試行)
4. ch/sh比分散は標本ノイズを超える (テストG: Z = 8.55)

### 攻撃に耐えなかった主張:
1. ch=地上部の「100%精度」は自明(ch全ページの99.1%に出現)
2. -yd-の4/4一致は多重検定で予想される偽陽性
3. ディオスコリデス比較にベースラインなし
4. 予測精度が事前登録されていなかった
5. 循環的方法論: 図版を分類した分析者が形態素意味を提案

## 第6部: 限界と今後の課題

### 限界

1. **確認バイアス**: 製薬記法仮説から出発し、代替仮説は同等に検証されていない
2. **循環論法リスク**: 図版を形態素意味提案者と同一人物が分類
3. **対照コーパス未実施**: 同一パイプラインを既知言語で実行していない（最重要の未実施実験）
4. **EVA転写依存性**: 単一転写に依存
5. **ブラインドテストのサンプルサイズ**: N = 9.5は個別有意性に不足

### 今後の課題（優先順位順）

1. **対照コーパス検証** -- 同一分析をトルコ語・フィンランド語・スワヒリ語コーパスで実行
2. **大規模事前登録ブラインドテスト** (N > 30) + 独立植物学者による図版分類
3. **Currier A/B分割の正式分析** -- 形態素分布が既知の言語分割を反映しているだけか検証
4. **15世紀イタリア製薬手稿での類似記法検索**
5. **生成モデル構築** -- 発見されたルールで生成し、実手稿と統計比較

## 結論

ヴォイニッチ手稿は解読されていない。本研究は解読を達成したとは主張しない。

しかし、以下を確立した:

1. テキストは真の構造を持つ -- ランダム生成や単純なグリルでは説明できない語間依存性と形態論的構造が存在する
2. その構造は自然言語よりも構成的記法体系に整合的である
3. 少なくとも1つの形態素(ty)が植物図版の視覚的特徴と有意に相関する
4. しかし、形態素に割り当てた意味の多くは検証に失敗した -- sh=根、cthy=果実、dchy=樹木状はすべて棄却された
5. 記述的命名体系の仮説は有望だが、陽性予測のみが機能し(79%)、陰性予測は失敗する

最も重要な方法論的貢献は、ヴォイニッチ研究への事前登録ブラインド予測テストの導入である。今後の研究はすべて、形態素意味の主張を公表する前にこのレベルの検証を行うべきである。

すべての失敗を報告する。これこそが誠実な学術研究を価値あるものにする。
