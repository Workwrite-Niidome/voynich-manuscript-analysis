## 要旨

ヴォイニッチ写本（バイネッキ稀覯本写本図書館 MS 408、放射性炭素年代測定1404--1438年）は1世紀以上にわたり解読に抵抗してきた。本稿は、20以上の分析サイクルにわたる反復的仮説生成・仮説破壊サイクル、248を超える分析ファイル、2回の事前登録済みブラインド予測テスト、専任のレッドチームレビュー、112枚の薬草ページ全面検証、薬学的符号化テスト、ページ内構造分析、計量システム検証を用いた、EVA転写全体の体系的計算分析の結果を報告する。

本研究の方法論的貢献として、ヴォイニッチ研究にブラインド予測テストを導入した。これは本分野で初めての試みである。テストされた23以上の仮説のうち12が排除され、検証済みの構造的知見、有望な仮説、反証された主張の明確な分離が生じた。

**検証済みの知見**（全テストに耐えた）：(1) 接尾辞選択における語間サンディ（10分割交差検証、42.9% vs. 34.7%ベースライン、p < 0.001、Cramer's V = 0.22）；(2) 6層の構成的形態論アーキテクチャ；(3) 語彙的特異性を符号化する母音グレード（r = -0.954）；(4) -dy/-ol接尾辞のクロスオーバーを伴うページ内テンプレート構造（第7-8行、カイ二乗 < 0.001）；(5) 薬草ページにおける93%の固有先頭語；(6) 倍増・位置・文脈の整合性を持つqo-k計量システム；(7) 線形葉植物の検証済み予測因子としてのty（Fisher p = 0.008, OR = 18.0）；(8) 頭痛ペアの語彙共有（99パーセンタイル、p < 0.01）；(9) ボンフェローニ補正を生き残った14/15の形態素部分文字列の非ランダム分布。

**反証された主張**：sh = 地下/根（ブラインドテスト25%）、cthy = 果実（全面検証 p = 0.66）、dchy = 高木（p = 1.0）、全面的な語-挿絵相関の失敗、ドラクマ:オンス比の不一致。

写本は未解読のままである。しかし、検証済みの知見は解空間を制約し、単純なでっち上げメカニズムとは両立しない真正の言語的または記法的構造を示している。

## 1. 序論

ヴォイニッチ写本は、イェール大学バイネッキ稀覯本写本図書館に所蔵される234ページのヴェラム冊子（MS 408）である。放射性炭素年代測定によりヴェラムの製造は1404年から1438年の間に位置づけられる。未知の文字で書かれ、未同定の植物、天文図、入浴場面の挿絵が施されており、1912年のウィルフリッド・ヴォイニッチによる再発見以来、集中的研究の対象となってきた。

過去の計算分析により、いくつかの堅牢な統計的性質が確立されている。Bennett（1976）とLandini（2001）はZipf則への適合を確認した。Amancio et al.（2013）は自然言語とランダムテキストの中間のエントロピー値を測定した。Stolfi（2005）とZattera（2023）は個々のグリフの位置スロット文法を同定した。Currier（1976）は2つの「言語」（AとB）を区別し、Davis（2020）は5人の筆写者の手を同定した。

### 1.1 方法論的逸脱

本研究は先行研究から5つの点で逸脱した。

1. **反復的仮説破壊**: 各分析サイクルは前サイクルの生存仮説の反証を試み、専任の敵対的レッドチームレビュー（サイクル16）で頂点に達した。

2. **事前登録済みブラインド予測テスト**: 各10葉の2ラウンドで、全予測を挿絵確認前に記録した。ヴォイニッチ研究における方法論的革新であり、過去の計算論的ヴォイニッチ分析でこのレベルの前向き検証を行ったものはない。

3. **全コーパス分析**: 選択された葉ではなく、完全なEVA転写（226葉にわたる37,779トークン）を分析した。

4. **多段階検証**: 30ページの訓練セットからの初期知見を112枚の薬草ページすべてで再テストし、上位5つの形態素-特徴関連のうち4つが全面検証で再現に失敗することを明らかにした。

5. **完全な失敗報告**: テストされた23以上の仮説すべてが方法・結果・状態とともに文書化されている。

### 1.2 主要な知的貢献

最も重要な概念的突破は、15世紀の著者が情報理論にアクセスできなかったことの認識であった。現代の暗号分析者は、符号化者が情報密度を最大化すると仮定する。しかし15世紀の薬剤師は、記憶しやすさと体系性のために最適化されたシステムを設計したであろう -- 暗号よりも化学命名法（IUPAC）に近い。

この再枠組みが、階層的語幹構造の発見に直接つながった。3,483の固有語幹が同音語統合による削減に抵抗した（5回の反復、最良結果：約1,500語幹 -- まだ多すぎる）とき、「メッセージを隠すためにどんな暗号が使われたか？」から「知識を組織するためにどんな記法システムが設計されたか？」への転換が、コードブックアプローチが失敗したところで成功した構成的語根分解を可能にした。

形式的原理：**15世紀の記法システムは情報理論的に完全であり得ない。したがって語幹は恣意的なコードではなく生成規則を持たなければならない。** この原理は他の未解読の歴史的文字体系にも適用可能かもしれない。

## 2. 検証済みの知見

以下の結果は、交差検証、順列検定、ボンフェローニ補正、敵対的レッドチームレビュー、ブラインド予測テスト、および/または全面的再現に耐えた。

### 2.1 語間接尾辞依存性（サンディ）

ヴォイニッチ写本の語は、ほぼ等確率の3つの子音 -- n（34.7%）、l（32.6%）、r（32.8%）-- のいずれかで終わり、その選択は文脈により体系的に条件づけられる。

**第1層 -- 語幹末母音**（最強効果、MI = 0.929ビット）。-iで終わる語幹は一貫して-nを取る（69の証明済み語幹の100%）；-oで終わる語幹は-lを好む（67語幹の79%）；-aで終わる語幹は-rを好む（48語幹の77%）。

**第2層 -- 後続語の語頭**（中程度の効果、Cramer's V = 0.215、カイ二乗 = 1,404、p = 1.88 x 10^-278）。後続語の語頭は3つのクラスに分割される：母音は先行する-rを2.0倍のリフトで好む；閉鎖音は-lを1.41--2.08倍のリフトで好む；摩擦音は弱い-n選好を示す。

**第3層 -- 韻律境界**。この効果は行内で作用するが行境界を越えると崩壊する（例：k語頭語の前で、-lは行内71.3%だが行間では22.2%のみ）。

**交差検証結果:**

| テスト | 結果 |
|------|--------|
| 10分割CV平均正確度 | 42.9% +/- 2.7%（全10分割が34.7%ベースラインを超過） |
| 順列検定（n=1000） | p < 0.001（0/1000が実際の正確度に到達） |
| 分割間確率相関 | Pearson r = 0.84（p = 5.36 x 10^-13） |
| 分割間トリガー一致 | 80%（15語頭中12） |
| 語間 vs. 語内リフト | +8.20 pp vs. +3.95 pp |

語間の条件づけは語内の約2倍の強さであり、転写上のアーティファクトではなく語境界特異的プロセスと整合する。この知見は単純なカルダングリル（Rugg 2004）では生成困難であり、でっち上げ仮説を実質的に弱体化させる。

### 2.2 形態論アーキテクチャ（6層）

ヴォイニッチの語は多スロット構造に分解される：

```
語 = [接頭辞] + [分類子 + 語根 + 母音グレード + 端末] + [接尾辞]
```

この6層アーキテクチャは、先行分析で報告された3項の接頭辞+語幹+接尾辞よりも精細である。重要な革新は、不透明な「語幹」として扱われてきたものの内部分解である。

**接頭辞目録**（15の生産的タイプ）：ch-（16.9%）、qo-（14.2%）、o-（9.2%）、d-（8.0%）、sh-（7.8%）、ot-（6.4%）、ok-（6.3%）、y-（4.4%）、その他7つ。

**接尾辞目録**（17タイプ）、行位置挙動別に組織化：終止子（-am, -m；行末73%）；開始子（-or, -eey, -ol；行頭71--78%）；格標識（-aiin, -ain, -ar, -ey）；述語（-y, -dy, -al）；接続子（-iin, -edy）。

**接頭辞-接尾辞の独立性**: NMI(接頭辞, 接尾辞) = 0.032、NMI(接頭辞, 語幹) = 0.508、NMI(語幹, 接尾辞) = 0.379と比較。このほぼゼロの結合は、自然形態論（ラテン語、イタリア語、トルコ語で接頭辞-接尾辞NMIは通常0.15--0.40）よりも多次元記法システムとより整合する特徴である。

### 2.3 母音グレード = 特異性（r = -0.954）

母音グレードは派生連鎖を形成する：k -> ke -> kee -> keeo -> keeod。3つの競合仮説がテストされた：

| 仮説 | テスト | 結果 | 状態 |
|------|------|------|------|
| H1: 植物部位（グレード = 葉/根/花/果実） | カイ二乗 | 51.53、パターンが予測と矛盾 | 棄却 |
| H2: 加工段階（生から加工へ） | カイ二乗 | 1,289、レシピ濃縮が単調 | 支持（交絡あり） |
| H3: 特異性（一般から特定へ） | Pearson r | -0.954、R二乗 = 0.910 | 強く支持 |

レシピセクションの濃縮はグレードとともに単調増加：裸 0.69倍 -> +e 1.66倍 -> +ee 2.82倍 -> +eeo 3.82倍。グレードシステムは派生的特異性階層として最もよく理解される：裸の語幹は一般的カテゴリを命名し、追加される各母音レベルは用語をより少ない文脈に制限する。

### 2.4 ページ内テンプレート構造

121枚の薬草ページの分析により体系的な位置構造が明らかになった。

**先頭行の固有性**: 薬草ページの93%（分類可能な先頭語を持つ56枚中53枚）が、薬草セクション全体で3回以下しか出現しない語で始まる。77%が薬草セクション固有の語で始まる。ランダムな語選択の期待率：約5--10%。これは第1行が植物名または識別子を符号化することと整合する。

**接尾辞クロスオーバー（最も強いページ内知見）**: 接尾辞-dyと-olは体系的なクロスオーバーを示す：
- -dyは10.4%（L1）で始まり3.1%（L15）まで減少 -- 定常的減少
- -olは7.4%（L1）で始まり15.7%（L15）まで増加 -- 定常的増加
- 2つの接尾辞はおよそ第7-8行で優勢を交代する

両分布とも統計的に有意（カイ二乗 < 0.001）。ランダムまたは無意味なテキストは平坦な分布を示すであろう。この接尾辞のクロスオーバーは、テキストが異なる行位置で異なる文法モードを持つ構造化された内容を符号化しているという強い証拠である。

**追加的位置シグナル**:
- pc-およびpo-接頭辞はほぼ第1行にのみ出現（2.8%および1.7% vs. 他ではほぼ0%）-- 植物命名機能と整合するline-1限定形態素
- sh-は第1行でピーク（12.4%）、第3行までに5.9%に低下
- cth-は第1行でほぼ不在（1.1%）、第2行で4.7%に上昇
- -am接尾辞は最後から2番目の行で急増（L14で5.4% vs. 他では0.8--1.7%）-- 節/項目の終止と整合

### 2.5 qo-k計量システム

qo-k語族（qokeey, qokeedy, qokey, qokedy, qokain, qokaiin, qokal, qoky, qokeeey）は以下の検証済み特性を持つ体系的パラダイムを形成する。

**確認された主張:**

| 特性 | 証拠 | 状態 |
|------|------|------|
| qo-kが計量を符号化 | レシピ集中、植物成分語が後続、位置パターン | 強い |
| qokeey = 標準/最頻単位 | 最高総頻度（340）、最高倍増（16） | 強い |
| qokeeey = 最大/最稀少単位 | 最低頻度（25）、グレード3 = 最大 | 強い |
| qoky = 汎用/必要量（quantum sufficit） | 最遅行内位置（0.666）、終止語が後続 | 強い |
| qokam = 用量終止標識 | 常に行末（位置1.000、n=5） | 強い |
| ok- vs. qo-kの区別（身体 vs. 数量） | 平行形態論、セクション分布の乖離 | 強い |

**文脈的検証**: 計量語の後に植物語彙（ch-、sh-接頭辞語）が高い割合で続き、期待される「[計量] [成分]」のレシピ構造を確認する。パターン「qokal ... dar」（5回）-- 「匙 + 与える」-- は標準的な薬学指示形式に一致する。

**分野横断的運用**: qokeeyは主に天文セクションに出現（201/340 = 59%）、ee-グレードが分野横断的に適用可能な「標準単位」を示すことを示唆する。

**失敗した主張**: ドラクマ:オンス頻度比は1.5:1（qokain = unciaならば期待値8:1）。特定のラテン語単位対応は「標準単位」と「必要量」以外は確認されない。

### 2.6 検証済み形態素-特徴関連: ty = 細い/線形（p = 0.008）

112枚の薬草ページ全面で検証された上位5つの形態素-特徴関連のうち、再現に成功したのは1つのみ：

| 形態素 | 主張される意味 | 原初カイ二乗（30ページ） | 全面Fisher p（112ページ） | 全面OR | 再現？ |
|--------|-------------|--------------------------|------------------------------|--------|-------|
| **ty** | **細い/線形** | **30.69** | **0.008** | **18.00** | **はい** |
| cthy | 果実/種子 | 60.44 | 0.661 | 1.29 | いいえ |
| dchy | 高い/樹木状 | 32.36 | 1.000 | 1.22 | いいえ |
| cphol | 繊維根 | 24.20 | 0.120 | 4.00 | 傾向のみ |
| dan | 分裂/分節 | 21.27 | 0.095 | 7.24 | 傾向のみ |

語tyはほぼ排他的分布を示す：分類済み出現5回のうち4回が線形葉ページ（PPV = 0.800、特異度 = 0.982）。

### 2.7 頭痛ペアの語彙共有（p < 0.01）

5つのアンカー植物を用いた薬学的符号化テストにおいて、f9r（ニゲラ、頭痛治療）とf47r（ブドウ、頭痛治療）のペアはランダムな言語Aの薬草ページペアの99.1パーセンタイルで語彙を共有する。

| 指標 | f9r-f47rペア | ランダムペア平均 | パーセンタイル |
|------|------------|----------------|----------:|
| Jaccard類似度 | 0.1226 | 0.0615 | 99.1 |
| 排他的共有語 | 8 | 2.1 | 99.8 |

8つの排他的共有語（chaiin, chdy, choy, cphy, cthaiin, cthey, shy, tchol）は他のアンカーページには見られない。言語タイプ（両方とも言語A）や写本内の近接性（f9rはカイヤB、f47rはカイヤF）では説明できない。

### 2.8 非ランダム形態素分布（14/15がボンフェローニ生存）

15の主張された形態素のそれぞれについて、葉ごとの頻度分散を1,000回のランダムシャッフルと比較した。14/15の形態素がZ = 3.15--29.51で生存し、6,384の可能な2--4文字部分文字列に対するボンフェローニ補正に耐えた。さらに、0/50のランダム試行（7つのランダムな頻出部分文字列を選択）が同等の分解率を達成せず、任意の頻出部分文字列セットが同等の結果を生むという可能性を排除した。

**重要な注意**: 非ランダム分布は意味的意味を証明しない。分布はCurrier A/B言語差異、筆写者のクラスタリング、トピカルな語彙変動から生じる可能性がある。

### 2.9 複合分類子の構成性（8/8）

8つの複合語根（kch, tch, pch, lch, lsh, ckh, cth, cph）すべてが、その構成要素から予測されるとおりに写本セクション間で分布した（8/8一致）。

## 3. 仮説（有望だが未検証）

### 3.1 記述的命名システム

形態論アーキテクチャ、構成性、挿絵との相関は、語が観察可能な特徴を符号化するシステムを示唆する。しかし証拠は混在している。

**支持証拠（ブラインドテスト）**: ブラインドテスト2で、肯定的予測（形態素存在 -> 特徴存在）は9.5予測にわたり約35%のチャンスベースラインに対して約79%の正確度を達成した。

| 肯定的予測 | 正確度 | vs. チャンス |
|-----------|--------|------------|
| cth存在 -> 果実/種子 | 83%（2.5/3） | +48% |
| dal存在 -> 丸い/球状 | 100%（2/2） | +70% |
| oiin存在 -> 花 | 100%（1/1） | +45% |
| ty高 -> 細い/線形 | 100%（1/1） | +75% |
| **合計** | **約79%** | **+44%** |

**反証**: 全体的なブラインドテスト正確度は67.6%（テスト1）と63.2%（テスト2）で、推定チャンスベースライン55--65%をわずかに上回るのみ。否定的予測は体系的に失敗する。

**状態**: 有望だが未検証。より大規模な事前登録済みテストが必要。

## 4. 反証された主張と失敗したアプローチ

失敗についての透明性は誠実な学問に不可欠である。20以上の分析サイクルで12以上の仮説が排除された。

### 4.1 sh = 地下/根：反証

**ブラインドテスト結果**: 肯定的予測は25%（0.5/2）；否定的予測は43.8%（3.5/8）。両方ともチャンス（可視根のベースレート約50%）以下。

**コーパス証拠**: sholは根が目立つ葉で0.99%、葉が目立つ葉で1.42%出現 -- 予測と逆方向（Mann-Whitney p ~ 0.5）。

**診断**: 主張はch-との構造的対称性（美的仮定であり、経験的ではない）に由来し、利用可能であった反証に対して適切にテストされなかった。これは確認バイアスの警告例として文書化されている。

### 4.2 cthy = 果実/種子：全面検証で反証

原初カイ二乗 = 60.44（9ページの訓練セット）。全面検証（112ページ、29の果実ページ）：Fisher正確 p = 0.661、OR = 1.29。原初効果は訓練セットの選択バイアスに駆動されていた。

### 4.3 dchy = 高い/樹木状：反証

原初カイ二乗 = 32.36（3ページ）。全面検証：Fisher p = 1.0、OR = 1.22。高い植物ページでの率（2.7/k）は低い植物ページ（2.3/k）と事実上同一。原初結果は3ページサンプルのアーティファクトであった。

### 4.4 否定的形態素予測：体系的失敗

両ブラインドテストにわたり、形態素が不在であることから視覚的特徴が不在であると予測することは、チャンス以下の成績であった：

| 予測タイプ | 正確度 | チャンスベースライン | 評価 |
|-----------|--------|-----------------|------|
| oiin不在 -> 花なし | 43%（テスト2） | 約45% | チャンス水準 |
| yd不在 -> 裂葉なし | 50%（テスト2） | 約65% | チャンス以下 |
| dal不在 -> 丸い特徴なし | 58%（テスト2） | 約70% | チャンス以下 |

### 4.5 全面的な語-挿絵相関：失敗

168の語-特徴組み合わせ（129枚の薬草ページで21の高頻度語 x 8つの視覚特徴）のうち、ボンフェローニ補正後に有意に達した相関は1つのみ -- sho vs. roots_visible（r = -0.334）-- であり、これは誤った方向（根がないときにshoが多い）であった。純粋なチャンスと整合する。

### 4.6 ドラクマ:オンス比の不一致

qokeey = ドラクマ、qokain = ウンチアとすると、頻度比は約8:1であるべき（中世の慣例）。観測値：227:156 = 1.5:1。

## 6. ブラインド予測テスト：方法論と結果

### 6.1 プロトコル

我々の知る限り、過去のヴォイニッチ分析で事前登録済みブラインド予測テストにその主張を晒したものはない。

1. 以前に分析されていない葉を選択（各テスト10枚）
2. 先頭語および/または全葉テキストを抽出
3. 主張されたシステムを使用して形態素に分解
4. 挿絵確認前にすべての予測を記録
5. 予測ロック後に挿絵を確認
6. 各予測をHIT、MISS、またはPARTIALとして採点
7. ヴォイニッチ薬草挿絵ベースレートから推定されるチャンスベースラインと正確度を比較

### 6.2 ブラインドテスト1（10葉：f20r--f38r）

- **全体正確度**: 67.6%（PARTIALを0.5として25/37）
- **推定チャンス**: 約60--65%
- **主要結果**: ch存在 -> 葉が顕著：4/4 = 100%；sh存在 -> 根が顕著：0.5/2 = 25%（**失敗**）
- **帰結**: sh = 根の主張が決定的に反証された。sh解釈の改訂を促した。

### 6.3 ブラインドテスト2（10葉：f7r--f19r）

- **全体正確度**: 63.2%（PARTIALを0.5として36/57）
- **推定チャンス**: 約55--60%
- **主要結果**: 肯定的予測：約79%正確度；否定的予測：約50%正確度（チャンス以下）
- **主要な非対称性**: 肯定的予測は真正のシグナルを示す；否定的予測は体系的に失敗する。

### 6.4 方法論的意義

これらのテストは回顧的分析には見えない洞察を明らかにした：

1. **肯定的予測は機能する；否定的予測は機能しない。** これはシステムを部分的記述子モデルに制約する。
2. **一部の形態素の意味は誤りである。** sh = 根は明快に反証され、改訂を促した。
3. **小Nの訓練セットは膨張した統計を生む。** 30ページの訓練セットからのカイ二乗値は5形態素中4で全面検証時に再現しなかった。
4. **ベースレートが重要。** 「葉が顕著」を予測することは常に「はい」と言うことで約70%の正確度を達成する。

将来のヴォイニッチ研究が形態素-意味の主張を公表する前に同様のブラインドテストプロトコルを採用することを強く推奨する。

## 8. 議論

### 8.1 写本が何でないか

| 理論 | 状態 | 証拠 |
|------|------|------|
| Rugg（2004）: 単純カルダングリルのでっち上げ | 部分的反証 | 3層の語間依存性を生成できない |
| Cheshire（2019）: 原ロマンス語 | 反証 | 2母音システム、17% A/B語彙重複と非互換 |
| Bax（2014）: 部分的音声 | 排除 | 全5アンカー名で音声回復失敗 |
| Ardic（2025）: 古トルコ語 | 排除 | 母音調和58%（チャンス水準） |

### 8.2 写本が何であり得るか

生存する証拠は以下の特徴を持つ**構築された記法システム**と最も整合する：
- 解読可能な外部フレーム（15接頭辞、17接尾辞）
- 約10の子音語根から構築される構成的語幹の内部コア
- 特異性階層（母音グレード）
- 語間の音韻的または記法的依存性（サンディ）
- 薬学的および天文学的領域にわたって機能する計量パラダイム（qo-k）
- 行位置間で文法的シフトを伴うページ内テンプレート構造

レッドチーム後の巨視的仮説の信頼度推定：

| 仮説 | 信頼度 |
|-----|-------|
| 構築された薬学的記法 | 35--45% |
| 冗長同音暗号（Naibbe型） | 25--35% |
| 未発見/記録不十分な自然言語 | 10--20% |
| 行認識を持つ洗練されたでっち上げ | 15--25% |

### 8.3 限界

1. **確認バイアス。** 分析は薬学的記法仮説から始まった。代替案は同等の厳密さでテストされなかった。
2. **循環推論のリスク。** 挿絵は形態素の意味を提案するのと同じ分析者により分類された。植物学専門家による独立したブラインド分類が必要。
3. **EVA転写依存。** すべての分析は単一の転写システムに依存し、アーティファクトを導入する可能性がある。
4. **対照コーパスの欠如。** 最も重要な未実施の実験：同等サイズの膠着語コーパスに対する同一パイプラインの実行。
5. **ブラインドテストのサンプルサイズ。** 肯定的予測結果（N = 9.5）は個別の有意性には小さすぎる。
6. **Currier A/B交絡。** 内容に帰属される語彙変動の多くが、十分に文書化されたA/B言語分割を反映している可能性がある。

### 8.4 ほとんどの形態素-特徴関連が再現に失敗した理由

原初の30ページ研究は3--9ページの特徴グループサイズを使用した。カイ二乗値は稀な語（6--82の総出現）を用いて語レベルで計算された。これは極端な不安定性を生んだ：

**例**: dchyは3ページ（195語合計）にわたるわずか4語の出現に基づきカイ二乗 = 32.36を持った。22の高い植物ページに拡張すると、率は2.7/kに低下 -- 「シグナル」は4語の揺らぎであった。これは稀な形態素を用いた小Nの言語学研究に対する警告の教訓である。

## 9. 今後の研究

1. **対照コーパス検証** -- 最優先。既知の言語コーパス（トルコ語、フィンランド語、スワヒリ語、グルジア語）に同一パイプラインを実行。
2. **大規模事前登録済みブラインドテスト**（N = 30以上）、提案された形態素の意味を知らない専門家による独立した植物学的挿絵分類を伴う。
3. **正式なCurrier A/B分割分析**: 形態素分布が既知の言語分割を単に反映しているかどうかを決定。
4. **より多くの植物同定が必要**: 薬学的符号化テストは5つのアンカー植物のみに制限されていた。言語AとBにわたりバランスの取れた15--20の確実に同定された植物が必要。
5. **多スペクトルイメージングデータ分析**: バイネッキ図書館の増大する多スペクトルイメージングデータを活用し、消去またはフェードしたテキスト、インク組成、可能な修正を同定。
6. **Davisの葉順変更の効果**: 共役葉が19%高い語彙一致を示す検証済みの知見（Jaccard 0.1070 vs. 0.0898）は、Davisの提案する順変更全体にわたる体系的調査に値する。
7. **特定の中世薬学文献との相互参照**: Antidotarium Nicolai、Circa Instans、Carrara Herbalは同じ歴史的時期と地理的地域を共有する。体系的な構造比較（レシピ形式、計量慣例、成分順序）が解空間を制約し得る。
8. **カルダングリルのサンディテスト**: 行認識グリルテキストを生成し、語間相関をテストして、洗練されたグリルが観測されたサンディパターンを再現できるかを決定。

## 10. 方法

### 10.1 コーパス

主要データソースはTakahashi--Zandbergen EVA転写（RF1b-e.txt）であり、37,779語トークン、8,500固有語型、226,468文字を226葉にわたり含む。

### 10.2 分析アーキテクチャ

調査はClaude Opus 4.6（1Mコンテキスト）により人間の研究者の指揮の下で実施された。20以上の分析サイクルが生成-テスト-破壊プロトコルに従って実行された。合計248以上のファイルが生成された。

**マルチエージェント主張に関する重要な注意**: すべてのエージェントは同一のモデルの同一の訓練データとバイアスを持つインスタンスであった。真のマルチエージェント独立性は根本的に異なる分析システムを必要とする。

### 10.3 統計的方法

Cramer's Vを伴うカイ二乗検定；相互情報量（MI）および正規化MI（NMI）；順列検定（n = 1,000）；交差検証（50/50分割および10分割）；PearsonおよびSpearman相関；Fisher正確検定（両側）；ボンフェローニ補正；Benjamini-Hochberg FDR；Mann--Whitney U検定；二項確率推定；K-meansクラスタリング；JaccardおよびCosine類似度。

### 10.4 ブラインド予測プロトコル

各10葉の2ラウンド、以前に分析されていないページから選択。すべての予測を挿絵確認前に記録。予測をロックしイェール大学バイネッキのデジタルファクシミリ挿絵に対して採点。チャンスベースラインを薬草セクションの視覚的特徴ベースレートから推定。

### 10.5 全面形態素検証

30ページの訓練セットからの上位5つの形態素-特徴関連を112枚の薬草ページすべてで再テスト。各ページを公表されたファクシミリスキャンから関連する視覚的特徴について分類。ページレベル（2x2 Fisher正確）および語レベル（Yates補正付きカイ二乗）の両分析を実行。感度、特異度、PPV、NPV、オッズ比を計算。

### 10.6 薬学的符号化テスト

既知の同定を持つ5つのアンカー植物について、(a) 投与経路、(b) ガレン的質（温/冷）、(c) 身体システム、(d) 調製方法、(e) sh-頻度 vs. ガレン的強度による語彙グルーピングをテスト。Jaccard/Cosine類似度、順列検定、Spearman相関、K-meansクラスタリング（k=3, 5）を適用。

### 10.7 計量システム検証

qo-k語族を7次元にわたり検証：(1) 行内語位置、(2) 葉正規化行位置、(3) 倍増頻度、(4) グレード-頻度単調性、(5) 中世規範との頻度比、(6) 文脈的後続語、(7) セクション分布（薬草/レシピ/天文）。
