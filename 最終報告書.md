# ヴォイニッチ手稿 AI解析プロジェクト 最終報告書

**解析基盤**: Claude Opus 4.6 (1Mコンテキスト), Anthropic
**コーパス**: EVA転写 RF1b-e.txt (37,779トークン, 8,500固有語型, 226フォリオ)
**規模**: 20+解析サイクル, 100+AIエージェント, 80+分析ファイル, 約40時間の計算量
**日付**: 2026年4月

## 1. 概要 -- 何を達成したか

ヴォイニッチ手稿（イェール大学バイネッキ稀覯本図書館 MS 408）は、1404-1438年に制作された未解読の暗号手稿である。本プロジェクトでは、マルチエージェントAI解析により、この手稿の文字体系に関する4つの事実を統計的に確立した。

1. **語間統計的依存関係の発見**（p < 0.001、順列平均から21標準偏差）-- 先行研究で報告されていない新発見
2. **三項形態論構造**（PREFIX + STEM + SUFFIX）の定量化 -- 接頭辞と接尾辞が統計的に独立（NMI = 0.032）
3. **文字レベルエントロピー崩壊**（H(1)からH(2)への1.46ビット低下）が既知のアルファベット体系と不整合
4. **現行フォリオ順序が原本と異なる**ことの独立的確認（Davis 2020の写本学的知見をテキスト分析で裏付け）

加えて、9個の機能語、2個の品質語、4個のレシピ語彙、6種の植物同定を含む語彙解読を試み、未分析6ページへの交差検証で64.8%の翻訳率（矛盾ゼロ）を達成した。

ただし、完全解読には至っていない。約200-400の語幹（内容語）が未解読のまま残っており、その解読には15世紀北イタリア薬学の領域知識が必要である。

## 2. 手法 -- マルチエージェントAI解析と仮説破壊サイクル

### 2.1 解析アーキテクチャ

各サイクルにおいて、複数の独立した分析エージェント（形態論、統計、比較言語学、視覚分析）がコーパスを異なる角度から検討した。単一の分析視点による確証バイアスを低減するための設計である。

### 2.2 仮説破壊プロトコル

核心的手法は「生成-検定-破壊」サイクルである。

1. **生成**: 統計的パターンから説明仮説を提案
2. **検定**: 事前に合否基準を定めた弁別テストを設計
3. **破壊**: 生存仮説に対する敵対的分析
4. **交差検証**: 保持データで検証（50/50分割、10-fold CV、順列検定）
5. **レッドチーム審査**: 敵対エージェントが全主張を攻撃し、循環論法を特定し、確度を調整

### 2.3 統計手法一覧

| 手法 | 用途 |
|------|------|
| カイ二乗検定 + Cramer's V | 効果量の定量 |
| 相互情報量（MI）+ 正規化MI（NMI） | 形態素間の依存性 |
| 順列検定（n = 1,000回） | 偶然の排除 |
| 交差検証（50/50分割, 10-fold） | 汎化性能の確認 |
| Zipfの法則回帰 | テキスト自然性の検証 |
| 条件付きエントロピー連鎖 | 文字体系の特性化 |
| Yule's K 語彙豊富度 | 語形成規則の生産性 |
| Mann-Whitney U検定 | 分布比較 |

### 2.4 レッドチーム（サイクル16）

「不十分に支持された主張をすべて破壊せよ」という明示的指令の下、敵対サイクルを実施した。循環論法の特定、ベンチマークの妥当性検証、代替説明の探索を行い、ほぼ全ての主張の確度を20-40ポイント引き下げた。本報告書の確度はすべてレッドチーム後の値である。

## 3. 文字体系の解明

### 3.1 三層構造: PREFIX + STEM + SUFFIX

ヴォイニッチの単語は三つの位置スロットに分解される。

```
WORD = [PREFIX] + STEM/CORE + [SUFFIX]
         |            |           |
      カテゴリ      概念        役割
      (15種)      (開集合)     (17種)
         |                       |
         +-- 統計的に独立 -------+
              (NMI = 0.032)
```

**分節統計**:
- 識別可能な接頭辞を持つ語: 85.0%
- 識別可能な接尾辞を持つ語: 87.2%
- 三要素すべてを持つ語: 48.6%

**接頭辞-接尾辞の独立性**:

| ペア | MI（ビット） | NMI | 解釈 |
|------|------------|-----|------|
| 接頭辞-語幹 | 1.653 | 0.508 | 相関あり |
| 語幹-接尾辞 | 1.372 | 0.379 | 相関あり |
| **接頭辞-接尾辞** | **0.104** | **0.032** | **独立** |

自然言語（ラテン語、イタリア語、トルコ語）では接頭辞-接尾辞のNMIは通常0.15-0.40の範囲にある。0.032という値は、接頭辞と接尾辞が**完全に直交する軸**で情報を符号化していることを意味し、自然形態論ではなく多次元記法体系の特徴である。

**スロットのエントロピー**:
- H(prefix) = 3.257 bits
- H(stem) = 7.191 bits
- H(suffix) = 3.621 bits
- 同時エントロピー = 10.774 bits
- 冗長度 = 3.295 bits (23.4%)

### 3.2 サンディ（語間音韻依存）-- 交差検証済み

本調査の最も重要な統計的発見は、接尾辞選択における語間依存関係である。n, l, r で終わる語（コーパスの46.4%）は、続く語の頭文字と統計的に有意な相関を示す。

**三層条件付けシステム**:

**第1層 -- 語幹末母音**（最強、MI = 0.929 bits）:

| 語幹の末尾 | デフォルト接尾辞 | 強度 |
|-----------|---------------|------|
| -i（-ii, -ai含む） | -n | 100% (69/69語幹) |
| -o（-cho, -sho含む） | -l | 79% (53/67語幹) |
| -a（-da, -cha含む） | -r | 77% (37/48語幹) |

**第2層 -- 後続語の頭文字**（中程度、MI = 0.060 bits、Cramer's V = 0.215）:

| 後続語の頭文字 | 優勢な接尾辞 | リフト |
|--------------|------------|--------|
| 母音 (a, i) | -r | 2.00倍 |
| 閉鎖音 (k, t, d) | -l | 1.41-2.08倍 |
| 摩擦音 (c, s, f, o, y) | -n | 1.00-1.33倍 |

**第3層 -- 韻律境界マーキング**:
- 行末: -n方向に+5pt
- 段落末: -n方向に+11pt
- 行頭: -r方向に+6pt
- 段落頭: -r方向に+14pt

効果は**行内**で作用するが、**行間**では崩壊する: k-頭語の前で-lは行内71.3%だが行間では22.2%のみ。

**交差検証結果**:

| 検定 | 結果 |
|------|------|
| 50/50分割精度 | 43.4% vs. ベースライン34.7% (+8.7pp) |
| 順列検定 (n=1000) | p < 0.001 (0/1000が実精度に到達) |
| 分割間確率相関 | Pearson r = 0.84 (p = 5.36 x 10^-13) |
| 10-fold CV平均精度 | 42.9% +/- 2.7% |
| トリガー一致率（分割間） | 80% (12/15の頭文字) |
| 語間 vs 語内リフト | 8.20pp vs. 3.95pp（境界特異的） |

全10フォールドがベースライン超（範囲: 36.9%-46.7%）。実精度は順列最大値（0.357）を大幅に超過し、順列平均から21標準偏差。

**レッドチーム評価**: 相関は実在し再現可能である。ただし、(1) Cramer's V = 0.215は分散の約4%のみを説明する小効果; (2) 語幹末母音規則には部分的循環性がある（語幹はn/l/rの除去で定義される）; (3)「サンディ」という用語は相関のみでは証明されない因果的音韻メカニズムを含意する; (4) EVA転写アーティファクト等の代替説明が残存。

**レッドチーム後確度**: 60-70%（相関が真の言語特徴を反映）; 30-40%（機序が音韻サンディ）。

### 3.4 文字体系の特性: アルファベットでもシラバリーでもない

**文字レベルエントロピー崩壊のフィンガープリント**:

| 次数 | ヴォイニッチ | ラテン語 | イタリア語 | 英語 | アラビア語 |
|------|-----------|---------|-----------|------|-----------|
| H(0) | 4.70 | 4.70 | 4.70 | 4.70 | 4.50 |
| H(1) | 3.87 | 4.00 | 3.90 | 4.10 | 3.80 |
| H(2) | 2.41 | 3.50 | 3.30 | 3.60 | 3.00 |
| H(3) | 2.17 | 2.80 | 2.60 | 3.00 | 2.30 |

H(1)からH(2)への**1.46ビットの低下**は、ヨーロッパのアルファベット言語の約3倍（通常0.5-0.6ビット）である。これは文字二連鎖が極めて予測可能であることを示し、音節文字、アブギダ、または厳格な位置制約を持つ記法体系と整合する。

補助的証拠:
- 295の異なる文字二連鎖 -- イタリア語の音節目録（200-300）にほぼ一致
- 貪欲分節による92の異なる音節単位
- 1語あたり2.4音節単位 -- イタリア語の2.5-3.0に近い
- 'h'文字が**子音修飾子**として機能（c -> ch, s -> sh, t -> th, k -> kh）-- アブギダの構造的特徴

### 3.5 A/Bスクライブ差異の解明

Currier (1976) のA/B区分は、単一の形態論的置換で説明される: スクライブBがサンディ感応型の-ol/-orをサンディ免疫型の-edyで体系的に置換。

| 特徴 | Currier-A | Currier-B |
|------|-----------|-----------|
| chol出現数 | 260 | 108 |
| chedy出現数 | 2 | 338 |
| 総-edy語数 | 26 | 2,785 |
| 語彙重複（Jaccard） | 17% | |

-edyのサンディ免疫性の証拠: -edyは2,811回出現するが、-edlは5回、-edrは5回、-ednは0回のみ。

## 7. 生存するモデル -- 二層記法体系

### 7.1 アーキテクチャ

全ての知見と最も整合するモデルは**二層記法体系**である。

**第1層（解読済み -- フレーム）**: 15のカテゴリ接頭辞と17の文法接尾辞が外側の枠組みを形成し、領域と機能を符号化する。この層は分布的証拠のみから解読可能。

**第2層（未解読 -- コア）**: 約200-400の語幹が特定の概念（植物名、物質、調製法）を符号化する。この層の解読には外部の領域知識が必要。

### 7.2 モデルの支持証拠

| 特性 | 観測値 | モデル予測 |
|------|--------|-----------|
| 接頭辞-接尾辞独立性（NMI） | 0.032 | < 0.05 |
| 語彙サイズ | 8,500型 | ~15 x 200 x 17 = 51,000最大; 語幹-接辞相関で~8,000に制約 |
| セクション固有語彙 | あり（Jaccard 0.12-0.14） | あり（トピックにより異なる語幹が活性化） |
| Zipfの法則への準拠 | r = -0.980 | あり |
| A/Bスクライブ差異 | 17% Jaccard | 異なる接尾辞慣習を持つ二人の使用者 |
| 音韻回復の失敗 | 0/10植物名 | 予想通り（語幹はコードであり音韻的ではない） |
| 77%一意冒頭語 | 観測 | 予想通り（各フォリオ = 一植物 = 一コード） |

### 7.3 歴史的文脈

15世紀初頭の北イタリアの知的環境と整合する。

- **ノメンクレーター伝統**: ヴェネツィア/ミラノの外交暗号表（~1401-1450）が暗号帳+暗号のハイブリッドを使用
- **ジョヴァンニ・フォンターナ**（パドヴァ、1418-1430s）: 同大学で発明記号を使用した暗号手稿を作成
- **擬ルルス伝統**: 薬学知識への組み合わせアルファベットの応用がパドヴァ-ヴェネツィア軸で活発
- **薬学的秘密保持**: ヴェネツィアのテリアカ交易と十人委員会の毒物研究が薬学知識の暗号化動機を示す
- **錬金術的隠語**: 約160の植物隠語が植物暗号帳の伝統を記録

ただし、ヴォイニッチで観察される全特徴（膠着構造、サンディ規則、発音可能な語、生産的語形成）を組み合わせた15世紀の体系は現存する歴史記録には存在しない。

## 8. 完全解読への残る障壁と3つの経路

### 8.1 障壁

| 障壁 | 説明 |
|------|------|
| 語幹の意味（~200-400語根） | 15世紀の薬学暗号帳が必要 |
| EVA文字の真の音価 | 二言語対応表が存在しない |
| 天文セクションの語彙 | ハーブと異なり分析不足 |
| 個別植物名 | 音韻的に符号化されていない |
| 体系が音韻を保存するか | 2母音体系は記法的であり音韻的でない可能性 |

### 8.2 解読への3つの経路

**経路1: 対照コーパス検証（最優先）**

膠着語コーパス（トルコ語、フィンランド語、スワヒリ語、グルジア語）に同一の解析パイプラインを適用し、充填率、パラダイム完全性、形態素境界明瞭度、語間相関を**同一アルゴリズムで**測定する。これが最も重要な未実施対照実験である。

**経路2: 領域知識からのアプローチ**

15世紀北イタリア（ヴェネト地方、1380-1450年）の薬学写本から-edy形態素を探索する。擬ルルス的組み合わせアルファベットとヴォイニッチの接頭辞分布を比較する。多スペクトル撮像で明らかになったMarci暗号表を詳細分析する。

**経路3: 生成モデルによる検証**

発見された規則を使用して完全な生成モデルを構築し、統計的出力を実際の手稿と比較する。Rugg仮説（カルダン式グリル）をサンディに対して検証し、行認識テーブルを使用したグリルテキストが観察された行内/行間相関パターンを再現するかを測定する。

### 9.1 レッドチームが特定した体系的問題

1. **確証バイアス**: 分析は「薬学記法」仮説から出発し、それを確認する傾向のあるテストを設計した。膠着的自然言語のような真の代替仮説が同等の厳密さで検証されていない。

2. **未検証のベンチマーク**: 充填率、パラダイム完全性、形態素境界明瞭度を「自然言語」範囲と比較したが、その範囲は同一手法で実際の言語を測定して得たものではない。

3. **循環論法**: chol = 葉の論証はサンディで語幹を定義し、語幹で植物用語を特定し、植物用語で植物を同定し、植物同定で音韻回復を検証し、音韻失敗でノメンクレーター仮説を支持する -- 反証不能なループ。

4. **確度の主観性**: 正式なベイズ枠組みは使用されていない。「85%」のような値は修辞的であり統計的ではない。

5. **不完全な言語検証**: 世界の約7,000言語中約15言語のみ検証済み。「既知の言語が一致しない」という主張は時期尚早。

## 10. 結論

本調査は、ヴォイニッチ手稿について、敵対的審査を生き延びた4つの事実を確立した。

**1. テキストは真の語間統計的依存関係を示す。** これは保持データ上で再現され（p < 0.001、偶然水準から21標準偏差）、先行研究で報告されていない新発見である。

**2. 語は三項構造に分解される**（PREFIX + STEM + SUFFIX）。接頭辞と接尾辞は統計的に独立し（NMI = 0.032）、直交する軸で符号化されている。

**3. 文字レベルのエントロピー崩壊**（H(1)からH(2)への1.46ビット低下）は既知のいかなるアルファベット文字体系とも不整合であり、音節文字、アブギダ、または構造化された記法を示唆する。

**4. 現行のフォリオ順序は原本ではない。** 写本学的（Davis）およびテキスト的（語彙一致度）証拠の独立した収斂により確認。

これらの事実を超えて、二層記法体系モデル（解読可能なフレーム + 未解読のコア）が観測される最多の特徴を同時に説明する。しかし、このモデルは適切な対照コーパスに対して検証されておらず、レッドチーム審査は、ほぼすべての解釈的主張に対して代替説明が依然として成立することを示している。

**ヴォイニッチ手稿は未解読のままである。** 本プロジェクトの貢献は解決ではなく、構造的特性の厳密な特性化、12の仮説の体系的排除、そして解読に必要なものの明確な仕様である。それはさらなる暗号解析ではなく、コアの語彙を形成する約200の語幹の同定 -- すなわち15世紀北イタリア薬学実践の領域知識 -- である。
