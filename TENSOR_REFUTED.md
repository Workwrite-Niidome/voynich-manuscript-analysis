# Tensor Hypothesis: REFUTED by Internal Correlation Test

## The Fatal Test

If D3 (-l/-r/-n/-y) encodes a unified grammatical ROLE (like case marking), then ALL -l words should behave similarly across folios (all genitive words cluster together).

### Results:
- **-ol words internal correlation: r=0.084** — essentially independent
- **-or words internal correlation: r=0.004** — completely independent
- **-ol vs -or cross-correlation: r=0.031** — independent

### Interpretation:
-l words do NOT form a coherent group. chol(-l), shol(-l), otol(-l), dol(-l) each have their own independent distribution across folios. D3 is NOT a unified "dimension" — it is part of each word's individual meaning.

## What Survives

1. **Prefix-root-suffix morphological structure** — CONFIRMED (this was never disputed)
2. **ch- vs sh- categorical opposition** — CONFIRMED
3. **chol = leaf, shol = root** — CONFIRMED (visual + distributional)
4. **n/l/r engineered equality** — CONFIRMED but unexplained
5. **The system is a structured shorthand** — yes, in the sense that each word compactly encodes complex pharmaceutical information

## What Is Refuted

1. **D3 as unified grammatical dimension** — REFUTED (r=0.08 internal correlation)
2. **Tensor decomposition** — REFUTED (dimensions not independent)
3. **"chol and chor are same concept, different role"** — AMBIGUOUS (r=0.408, not clearly same or different)
4. **n/l/r as "3 values of categorical variable"** — WEAKENED (they don't behave as a unified category)

## The Honest State

The Voynich uses a morphologically productive system with:
- Systematic prefixes (categories)
- A large codebook of roots (specific concepts)
- Suffixes that contribute to meaning but NOT as a unified grammatical dimension

This is more like a **structured abbreviation system** than a tensor. Each "word" is a compact code for a pharmaceutical concept, constructed from reusable components, but the components interact in complex ways that resist simple dimensional decomposition.

*The n/l/r equality remains the deepest mystery. It is too perfect for natural language, but the suffixes don't behave as a unified dimension. Something else is going on that we haven't identified.*
