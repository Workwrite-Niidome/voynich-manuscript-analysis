# Undiscovered Language Candidate Search for the Voynich Manuscript

## Target Features (from prior analysis)

The Voynich manuscript text exhibits the following structural features that any candidate language must explain:

1. **Three-vowel system**: /a, i, o/ (typologically unusual -- /a, i, u/ is standard)
2. **Prefix + Root + Suffix morphology**: Words consistently decompose into three parts
3. **Sandhi-like phenomena**: Suffix choice is conditioned by stem-final vowel AND initial sound of the following word
4. **Specific suffix rules**: -i stems take -n (100%), -o stems take -l (79%), -a stems take -r (77%)
5. **Agglutinative structure**: Morphemes attach without significant fusion
6. **Consistent word-length distribution**: Most words are 4-8 characters
7. **Low entropy**: Unusually predictable text for a natural language

## 1. Romani (Romany) Language

### Historical Presence in 15th-Century Italy
- Roma populations migrated from the Balkans into Western Europe during the 14th-15th centuries
- Sinti Romani speakers reached northern Italy by the 15th century
- Significant contact with Venetian, Friulian, and Italian populations
- Roma were present in the Venice/Padua region during the manuscript's probable creation period

### Morphological Features
- **Agglutinative**: YES, partly. Romani nominal morphology is inflectional with significant agglutination, especially in Layer II case marking (postposed elements that became suffixes)
- **Prefix + Root + Suffix**: PARTIAL. Primarily suffixing. Verbs have two stems (present/perfective) with suffixal marking. No productive prefixing system
- **Layer system**: Three layers of case morphology -- Layer I (Old Indo-Aryan endings), Layer II (agglutinative postposed case markers), Layer III (adpositions)
- **Case suffixes**: -ke/-ge (dative), -te/-de (locative), -tar/-dar (ablative), -sa/-ha/-ca (instrumental), -ker/-ger (genitive)

### Phonological Features
- **Vowel system**: 5 cardinal vowels /i, e, a, o, u/ -- does NOT match Voynich 3-vowel system
- **Sinti Romani**: 6 vowels + 4 diphthongs -- even further from the target
- **Aspirated consonants**: Retains /ph, th, kh/ from Indo-Aryan heritage (unique in European context)
- **Retroflexion**: Lost (unlike Sanskrit/Hindi)
- **Sandhi**: No documented sandhi rules. Some morphophonological alternations (s/h alternation in certain suffix combinations), but nothing resembling the Voynich stem-vowel-conditioned suffix system

### Sanskrit Connection
- Romani is Indo-Aryan and related to Sanskrit/Hindi, but:
  - Lost retroflexion
  - Lost complex sandhi system
  - Lost three-vowel base (expanded to 5 vowels under European contact)
  - Retained aspiration distinction (unusual for Europe)

### Verdict: WEAK CANDIDATE
- **Matches**: Partial agglutination; Indo-Aryan heritage; historical presence in NE Italy
- **Mismatches**: 5-vowel system (not 3); primarily suffixing (no prefix+root+suffix); no sandhi; no vowel-conditioned suffix selection
- **Score**: 2/6 features matched

## 2. Dalmatian Romance Language

### Historical Context
- Spoken along the Dalmatian coast (modern Croatia), directly across the Adriatic from Venice
- Venice controlled much of the Dalmatian coast in the 15th century
- Ragusan Dalmatian survived until ~17th century; Vegliote until 1898
- Documented primarily by Matteo Bartoli from last speaker Tuone Udaina (d. 1898)

### Morphological Features
- **Agglutinative**: NO. Fusional Romance morphology
- **Prefix + Root + Suffix**: Standard Romance verbal conjugation patterns
- **Notable feature**: Collapse of morphological tense distinctions in Vegliote -- unusual simplification
- **Future tense**: Unique development different from other Romance languages

### Phonological Features
- **Vowel system**: 5 basic vowels /a, e, i, o, u/ -- standard Romance, does NOT match
- **Palatalization**: Unlike other Romance languages, Dalmatian did NOT palatalize /k, g/ before front vowels (conservative feature)
- **No sandhi**: Standard Romance phonology

### Verdict: VERY WEAK CANDIDATE
- **Matches**: Historical connection to Venice
- **Mismatches**: 5 vowels; fusional (not agglutinative); no sandhi; no prefix system; no vowel-conditioned suffixes
- **Score**: 0.5/6 features matched (only geographic proximity)

## 3. Rhaeto-Romance (Friulian / Ladin / Romansh)

### Historical Context
- Friulian spoken in NE Italy (Friuli region), adjacent to Venice/Padua
- Ladin in the Dolomites, Romansh in Switzerland
- First Friulian text ~1300; documented literary tradition by 15th century
- Geographic proximity to Voynich's likely origin area is strong

### Morphological Features
- **Agglutinative**: NO. Fusional Romance morphology
- **Prefix + Root + Suffix**: Standard Romance patterns
- **Distinctive features**: Retention of Latin C+l clusters; sigmatic noun plurals (like French -s, unlike Italian); sigmatic 2nd person singular

### Phonological Features
- **Vowel system**: Rich -- long/short vowel distinction is phonemic (unique among Romance languages)
  - Minimal pairs: "mil" (thousand) vs. "mil" (honey), "pes" (fish) vs. "pes" (weight)
  - Some dialects realize long vowels as diphthongs
- **Nasal vowels**: Present in Friulian (unlike Italian)
- **Palatal plosives**: /c/ and /dz/ present
- **Final devoicing**: Like German
- **No sandhi**: No cross-word phonological rules of the Voynich type

### Verdict: WEAK CANDIDATE
- **Matches**: Strong geographic proximity; some unusual phonological features
- **Mismatches**: Rich vowel system (not 3); fusional; no agglutination; no sandhi; no prefix system
- **Score**: 1/6 features matched (geography only)

## 4. Old Venetian / Venetan

### Historical Context
- NOT the same as standard Italian -- a distinct Romance variety
- Venice/Padua variant was the literary standard in 15th-16th century
- Lingua franca of Venetian maritime empire
- Strongest geographic candidate for the manuscript's origin

### Morphological Features
- **Agglutinative**: PARTIAL. Notable feature: agglutination of auxiliary "have" with clitic "ghe"
- **Compulsory clitic subject pronouns**: Venetian requires clitic pronouns (te, el/la, i/le) before verbs in 2nd/3rd person -- a "semi-analytical" verbal system unique among Romance languages
- **Past participle**: Invariable (unlike Italian)
- **Reflexive auxiliary**: Uses "aver" (have) like English/German, not "essar" (be) like Italian
- **No doubled consonants**: Unlike Italian

### Phonological Features
- **Vowel system**: 7 vowels -- does NOT match Voynich 3-vowel system
- **Consonant lenition**: Extensive intervocalic weakening
- **Apocope and syncope**: Restricted
- **No sandhi**: Standard Romance phonology

### Unique Features Not in Italian
- Compulsory clitic subject pronouns (semi-analytical verb)
- No geminate consonants
- Different auxiliary selection for reflexives
- Extensive consonant lenition

### Verdict: WEAK CANDIDATE
- **Matches**: Strongest geographic fit; some semi-agglutinative features (clitic fusion)
- **Mismatches**: 7 vowels; no sandhi; no prefix+root+suffix; no vowel-conditioned suffix selection
- **Score**: 1.5/6 features matched

## 5. Slavic Minority Languages in NE Italy (Slovenian)

### Historical Context
- Slovenian communities existed in Friuli from the 9th century onward
- 1458 Videm manuscript contains Slovene glosses in Friulian/Italian text -- confirms contact
- Northern Friuli gradually Germanized by 15th century, but Slovene communities persisted
- Major phonological changes occurring in 15th-century Slovene (stress shifts)

### Morphological Features
- **Agglutinative**: NO. Highly fusional Slavic morphology
- **Dual number**: Archaic feature retained (rare in Slavic languages)
- **Rich case system**: 6 cases, but fusional (not agglutinative)

### Phonological Features
- **Vowel system**: Rich -- multiple vowel qualities with distinctive length
- **Stress system**: Free-mobile, with accent shifts occurring in 15th century
- **No sandhi**: No cross-word phonological rules of Voynich type

### Verdict: VERY WEAK CANDIDATE
- **Matches**: Geographic presence in NE Italy
- **Mismatches**: Fusional; rich vowel system; no sandhi; no prefix+root+suffix; no vowel-conditioned suffixes
- **Score**: 0.5/6 features matched

## 6. Armenian

### Historical Context
- Armenian merchant community in Venice established by late 15th century (~1497 documented)
- Armenian communities in Italy from 12th-13th century (Cilician Armenia trade with Venice, Genoa, Pisa)
- Church of Santa Croce built in 15th century in "Calle degli Armeni"
- Well-organized community with hospice and church by 16th century
- San Lazzaro degli Armeni island later became famous Armenian monastery

### Morphological Features
- **Agglutinative**: YES -- strongly. Modern Armenian morphology is "typologically close to Turkish" with agglutinative declension
- **Prefix + Root + Suffix**: PARTIAL.
  - Suffixes dominate: case markers, possessive markers, verbal endings
  - Western Armenian prefixes "ge-" to form present tense (unique prefixal element)
  - Plural infix "-ner-" inserted between stem and ending (for polysyllabic nouns)
  - Up to 8 morphemes per word
- **Postpositions**: Uses postpositions instead of prepositions (like Turkish)
- **Passive and causative**: Morphological forms for all verbs

### Phonological Features
- **Vowel system**: 6 vowels in Eastern Armenian (a, e, i, o, u, schwa); 7+ in Classical Armenian; 8 in Western Armenian
  - Does NOT match Voynich 3-vowel system
  - However, if schwa is ignored and front/back pairs collapse, a reduced system might emerge in certain cipher contexts
- **No distinctive sandhi**: Standard Indo-European phonology
- **Theme vowels**: Play morphological role in Western Armenian verbs

### Verdict: MODERATE CANDIDATE
- **Matches**: Agglutinative morphology; presence in Venice; some prefix+suffix structure; postpositional system
- **Mismatches**: 6-8 vowels (not 3); no sandhi; no vowel-conditioned suffix selection; prefix system is limited
- **Score**: 2.5/6 features matched

## 7. Georgian

### Historical Context
- Georgian merchants could reach Italy via Constantinople (Trebizond-Constantinople trade route)
- No documented Georgian community in 15th-century Venice (unlike Armenian)
- Georgian contact with Italy was indirect, primarily through Byzantine intermediaries
- Weakest historical connection of all candidates

### Morphological Features
- **Agglutinative**: YES -- strongly, bordering on polysynthetic
- **Prefix + Root + Suffix**: **YES -- this is Georgian's defining feature**
  - Verb template: preverb + person prefix + version marker + ROOT + passive marker + thematic suffix + causative + imperfective + mood + auxiliary + person suffix + plural
  - Up to 8+ morphemes per verb form
  - Both prefixes AND suffixes are productive and obligatory
  - Root can be as short as 1-2 phonemes (e.g., -gh- for "open/receive/take")
- **Directionality prefixes**: mo- (coming), mi- (going), ga- (out), etc.
- **Version markers**: Preradical vowels that modify argument structure

### Phonological Features
- **Vowel system**: 5 vowels /a, e, i, o, u/ -- close to Voynich if /e/ and /u/ are marginal
  - No vowel length distinction
  - No nasalization
  - Simple, stable vowel system
- **Consonant clusters**: Extremely complex (up to 6+ consonants in a row)
- **Ejective consonants**: Three-way stop contrast (voiced/voiceless aspirate/ejective)
- **Sandhi**: LIMITED. Georgian is described as agglutinating where "affixes usually do not merge with each other or affect each other phonologically" -- this means LESS sandhi, not more
- **Syncope**: Root vowels can be lost in certain morphological contexts (e.g., ch'am > a-ch'm-ev)

### Critical Assessment
Georgian's prefix+root+suffix structure is the BEST match of any candidate for the Voynich word-building pattern. However:
- 5 vowels, not 3
- No documented sandhi of the stem-vowel-conditioned type
- No historical presence in Venice
- The consonant cluster complexity is unlike Voynich (which has relatively simple phonotactics)

### Verdict: INTERESTING BUT PROBLEMATIC
- **Matches**: Strong prefix+root+suffix; agglutinative; relatively small vowel system; short roots
- **Mismatches**: 5 vowels (not 3); no sandhi; no Venetian connection; consonant clusters unlike Voynich
- **Score**: 3/6 features matched (strongest morphological match)

## 8. Basque

### Historical Context
- Some Basque presence in Mediterranean trade (shipping, whaling)
- No documented Basque community in 15th-century Venice
- Basque sailors and merchants operated primarily in Atlantic/Bay of Biscay
- Pre-Indo-European isolate with no known relatives

### Morphological Features
- **Agglutinative**: YES -- primarily suffixing, with some ancient prefixes
- **Prefix + Root + Suffix**: YES, especially in verb morphology
  - Verb template: prefix (person/tense) + root + suffix (aspect/person/number)
  - Person prefixes: n- (1sg), h- (2sg informal), g- (1pl), z- (2sg formal/2pl)
  - Tense prefixes: d- (present), z- (past), l- (hypothetic), b- (imperative)
  - Participle formation: e-/i- prefix + root + -i/-n suffix
  - Up to 3 arguments indexed morphologically on single verb form
- **Ergative-absolutive alignment**: Fundamentally different from any known European language
- **Noun morphology**: Exclusively suffixing (no noun prefixes)

### Phonological Features
- **Vowel system**: 5 vowels /a, e, i, o, u/ -- same as Spanish
  - No vowel length distinction
  - Zuberoan dialect adds /y/ (front rounded)
  - Stable, simple system
- **Epenthetic vowels**: Vowel /e/ is inserted between consonant-final stems and consonant-initial suffixes
- **Sandhi**: LIMITED. Some morphophonological alternations at boundaries (vowel insertion, stem allomorphy), but no systematic cross-word sandhi of the Voynich type
- **No vowel-conditioned suffix selection**: Suffixes are determined by grammatical function, not by stem-final vowel

### Verdict: MODERATE CANDIDATE
- **Matches**: Agglutinative; prefix+root+suffix in verbs; pre-Indo-European (could explain unusual features); simple vowel system
- **Mismatches**: 5 vowels (not 3); no cross-word sandhi; no vowel-conditioned suffixes; weak Venetian connection; ergative alignment is not visible in Voynich
- **Score**: 2.5/6 features matched

## Key Findings

### 1. No Single Candidate Matches All Features
None of the eight tested languages matches all six key Voynich features. The most critical gap is universal: **no tested language has the vowel-conditioned suffix selection** (-i -> -n, -o -> -l, -a -> -r) that characterizes Voynich text.

### 2. The Three-Vowel System Remains Unexplained
Every European candidate has 5+ vowels. The only natural languages with 3-vowel systems (/a, i, u/) are geographically remote (Arabic, Quechua, Australian Aboriginal). The Voynich's apparent /a, i, o/ system remains typologically anomalous and may indicate:
- A cipher or notation system that collapses a richer vowel inventory
- A constructed language
- A phonological transcription by a speaker who perceived only 3 distinct vowel categories

### 3. Georgian Has the Best Morphological Match
Georgian's prefix + root + suffix verb template is the closest structural parallel to Voynich word-building. The pattern of short roots (as small as 1 phoneme) with multiple productive prefixes and suffixes creating 4-8 character words mirrors Voynich word-length distributions. However, Georgian lacks the sandhi features and has no documented connection to 15th-century Venice.

### 4. Armenian and Basque Are Worth Further Investigation
- **Armenian**: Agglutinative morphology + documented presence in Venice + the ge- prefix. If Classical Armenian's vowel system were reduced in a cipher context, some features align. Armenian merchants had reason to encode knowledge.
- **Basque**: Prefix+root+suffix in verbs + agglutinative + pre-Indo-European origins could explain unusual features. The "proto-Basque hypothesis" has been proposed by others (notably Cheshire 2019, though widely criticized).

### 5. The Voynich Feature Set Is Cross-Linguistically Unprecedented
The combination of:
- 3-vowel system
- Prefix + root + suffix agglutination
- Cross-word sandhi conditioned by stem-final vowel
- Specific n/l/r suffix distribution

...does not exist in ANY known natural language. This strongly supports either:
- A **constructed notation system** (pharmacological shorthand, alchemical code)
- A **cipher** that transforms a natural language into something with these statistical properties
- A **hoax** designed to look like language without encoding meaning

### 6. Previously Eliminated Languages Remain Eliminated
Standard Italian, Turkish, Arabic, Hebrew, and Latin were previously tested and eliminated. This expanded search confirms that the Voynich feature set is not found in any of the remaining plausible 15th-century Venetian-sphere languages either.

## Recommendations for Next Steps

1. **Re-examine the cipher hypothesis**: If the text is a cipher over Italian/Latin/Hebrew, the "3-vowel system" might be an artifact of the encipherment method, not a property of the underlying language
2. **Test Caucasian languages more deeply**: The Northwest Caucasian languages (Abkhaz, Ubykh, Kabardian) have genuinely small underlying vowel systems (2-3 vowels) with surface vowels determined by consonant context -- this could explain the Voynich pattern if a Caucasian substrate exists
3. **Investigate Persian/Dari**: Persian merchants were present in Venice; Middle Persian had morphological features not yet tested
4. **Consider hybrid systems**: A speaker of an agglutinative language (Georgian, Armenian, Basque) using a reduced vowel notation for a Romance language could produce exactly the Voynich pattern
