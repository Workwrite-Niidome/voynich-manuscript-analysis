# Validated Translations of Three Voynich Pages

## Methodology

All translations below use ONLY the validated morpheme decomposition system:
- **PREFIX** + [**CLASSIFIER** + **ROOT** + **GRADE** + **TERMINAL**] + **SUFFIX**
- Confidence levels assigned per-word based on the strength of evidence

### Confidence Marking System
- **[CERTAIN]** = validated at p<0.01 (morpheme structure, grade system, line position template)
- **[LIKELY]** = strong distributional evidence but not blind-tested
- **[POSSIBLE]** = hypothetical, based on frame position or analogy only
- **[?]** = unknown, only morpheme decomposition available

### Validated Tools Applied
| Tool | Confidence | Description |
|------|-----------|-------------|
| Morphological decomposition | 95% | PREFIX+[CLASSIFIER+ROOT+GRADE+TERMINAL]+SUFFIX |
| Function words (daiin, aiin, chey, ar) | 80% | Conjunction, preposition, copula, directional |
| Template (L1-7 description, L8+ prescription) | 85% | -dy dominant early, -ol dominant late |
| Line 1 = plant identifier | 95% | 93% unique words in L1 |
| qo- = measurement system | 85% | qokeey = standard unit |
| -am = clause terminal | 90% | Sentence/clause boundary |
| -ol/-or = subject/topic marker | 90% | With sandhi variation |
| -y/-dy = predicate/state marker | 85% | Descriptive function |
| -aiin = genitive/relational | 85% | Possession/relation |
| Grade: more vowels = more specific | 95% | r=-0.954 |
| ch- vs sh- domains | 75% | Aerial (herb/leaf) vs underground (root) |
